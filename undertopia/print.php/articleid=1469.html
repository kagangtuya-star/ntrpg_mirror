新法术（语言=V；姿势=S；材料=M；器材=F；法器=D；）

放逐术（Banish）
使受术者成为灵体
变化系
等级：术士/法师8
法术成分：语言, 姿势
施法时间：标准动作
距离：近距(25英尺+5英尺./每2等级)
目标：单一生物
持续时间： 1轮/每等级
豁免鉴定: 意志，通过则无效
法术抗力: 可

描述
该法术使受术者成为灵体，在法术效果期间都无法以任何方式与物质世界相接合。

法术效果
受术者在法术期间变得隐形而朦胧。对施法者而言仍然可见，但依然只有用能影响到灵体的法术才可以影响她。这种变成灵体的效果和幻化灵体法术效果相同，只是具有了上述魔兽争霸中的额外法术效果。


剑刃风暴（Blade storm）
施法者创造出两把魔法长剑
变化系
等级: 术士/法师 3
法术成分: 言语、姿势、材料
施法时间: 标准动作
距离: 个人
持续时间: 1轮/每等级 (解消)
豁免鉴定: 无
法术抗力: 无

描述
施法者魔术般地制造出一对锋利的剑，并且就像一个剑术的修行者一般暴风般地挥舞着它们。

法术效果
施法者制造出两把长剑，这两把长剑视同具有施法者每3等级+1的附魔值（最大+5）。在接下来的回合中，每等级1轮，施法者可以使用一个全力攻击动作，放弃他的所有正常攻击，改用施法者的全部基本攻击加值来对5英尺内的所有敌人进行一次近战攻击。若施法者没有在任一轮中进行一次全力攻击，则效果中止。
若施法者失去了其中一把或全部的刃器，只要法术依然作用，它们会在其下一轮中重新出现在施法者的手中。在法术持续期间，施法者视同精通军用武器长剑。

材料成分s:
两把小刀，双手各一把。在施法时分别持握在施法者手中，在法术结束时消失。


暴风雪（Blizzard）
无数冰尘、雪花和冰雹向着施法者的敌人头上砸去。
塑能系[寒冷]
等级: 术士/法师 3 法术成分: 言语、姿势、材料
施法时间: 标准动作
距离:远距 (400 英尺 + 40 英尺/每等级)
影响区域: 半径20英尺, 高40 英尺的柱状范围
持续时间: 专注，至多1轮/每2等级
豁免鉴定: 无
法术抗力: 可

描述
该法术召唤出无数雹子和冰片向施法者的敌人头上轰去。

法术效果
冻结的冰尘，雪花和冰雹倾斜而下，每回合对范围内造成1d6点冲击伤害以及1d6点寒冷伤害。施法者可以专注以维持法术至多5轮。

材料成分
一撮灰尘和几滴水。


血之活力（Blood lust）
英灵们给予受术者额外的攻击以及力量+4
变化系
等级: 萨满3，术士/法师 4
法术成分: 言语、姿势、材料
施法时间: 标准动作
距离: 近距 (25 英尺 + 5 英尺/每2等级)
目标：单一生物
持续时间:1轮/每等级
豁免鉴定: 强韧、通过则无效 (无害)
法术抗力: 可 (无害)

描述
施法者在受术者身上召唤出愤怒的英灵们，在一段时间内提升受术者的力量和反应。

法术效果
受影响的生物的行动和移动都比平常大大加快，额外的速度可以让其每轮多攻击一次。在他的轮中，他可以决定在常规动作之前或之后进行他的额外攻击。（只限于普通攻击，不可使用本次攻击进行其他动作，如施展法术等。）
受影响的生物同时也会获得额外的力量，法术会给予受术者力量+4的附魔加值，这可以使他的近战攻击、近战伤害以及其他使用力量修正的动作获得额外奖励。
该法术可以被缓慢术反制或解消。

材料成分:
一滴混杂着泥土的血液。


食尸（Cannibalize）
施法者通过吞食尸体来回复生命值。
死灵系[邪恶]
等级: 死灵2
法术成分; 言语、姿势
施法时间: 专注，至多4轮
距离: 个人
目标：一具尸体
持续时间: 立即
豁免鉴定: 无
法术抗力: 无

描述
施法者获得了一些食尸鬼的特征，尤其是它骨裂的下颌以及长长的舌头。施法者可以从尸体上吞食肉块以回复健康。

法术效果
由于这个咒语只对死去的生物有效且只治疗施法者，因此它没有豁免。尸体必须是完整同时具有真正肉体的（因此对胶质怪之类的生物无用）。
在吞食尸体肉块的过程中，施法者每维持专注1轮都可以获得1d8的生命值，至多回复至其最大生命值。尸体一旦被吃食过，就只会剩下很少的肉块，且不再视为完整的。一具尸体只能成为该咒语的目标一次。

腐臭蜂群（Carrion Swarm）
召唤大量的蜜蜂和害虫攻击敌人
咒法 (召唤)
等级: 术士/法师 5
法术成分: 言语、姿势、材料
施法时间: 标准动作
距离: 近距 (25 英尺. + 5 英尺/每2等级)
影响区域: 锥型区域
持续时间: 立即 豁免鉴定: 反射，通过则效果减半
法术抗力: 可

描述
以你的手指为中心延伸出一锥型区域，其中充满了腐臭的蜂群, 有毒的蜘蛛, 乱飞的甲虫, 和各种其他的害虫。

法术效果
这些蜂群将造成每施法者等级1D6的伤害（最高15D6），这些蜂群的攻击是非魔法效果的。对毒免疫的生物将承受一半的伤害（反射通过只受1/4的伤害）。伤害减免同样对本法术有作用。

材料成分:
一只蜘蛛、甲虫或者其他害虫的尸体。.

致残术（Cripple）
目标的攻击、伤害、反射豁免、跳跃以及力量将承受减值。
变化系
等级: 术士/法师 2
法术成分: 言语、姿势
施法时间: 标准动作
距离: 近距 (25 英尺. + 5 英尺/每2等级)
目标: 单个生物
持续时间: 1轮/每等级
豁免鉴定: 意志，通过则无效
法术抗力: 可

描述
在法术的持续时间内受影响的生物肢体的衰弱，使其移动及攻击速度都将呈急剧下降的趋势。

法术效果
目标生物在每轮中只能进行一个部分动作。另外其AC、近战攻击、近战伤害以及反射豁免上都将承受-2不利。受影响生物的跳跃距离只有正常时的一半。
除此之外，受影响的生物还将承受1d6的力量减值，以及每2个施法者等级额外的1点减值（额外减值最多为-5）。受影响生物的力量数值不会低于1。
致残术可以用加速术反制或解消。


死亡缠绕（Death Coil）
对有生命的敌人造成负能量伤害，对不死生物则为治疗效果。
死灵系
等级: 医师3, 死灵 3 法术成分: 言语、姿势
施法时间: 标准动作
距离: 近距 (25 fl + 5 英尺./ 每2等级))
目标: 单个生物
持续时间: 立即
豁免鉴定: 意志，通过则减半（见说明）
法术抗力: 可

描述
一股盘旋着的负能量从施法者掌中飞出，并准确无误地击中目标。

法术效果
这致命的缠绕将对有生命的敌人造成2d8（每施法者等级再+1，最大+10）的负能量伤害：一次成功的意志检定可使伤害减半。
但也因为不死生物的力量本就属于负能量，此法术将会给予他们同等程度的治疗，而非伤害。

高等死亡缠绕（Death Coil, Greater）
等级: 医师4, 死灵 4

与死亡缠绕相同，除了效果变为造成3d8（每施法者等级再+1，最大+15）的伤害。

次极死亡缠绕（Death Coil, Lesser）
等级: 医师2, 死灵 2

与死亡缠绕相同，除了效果变为造成1d8（每施法者等级再+1，最大+5）的伤害。


死亡契约（Death Pact）
摧毁一个不死生物，将其生命值赋予施法者。
死灵系列[邪恶]
等级: 死灵 5
法术成分: 言语、姿势
施法时间: 标准动作
距离:接触
目标: 单个不死生物
持续时间: 立即
豁免鉴定: 意志，通过则无效
法术抗力: 可

描述
一个在你控制下的不死生物——或许这是被你“复活亡灵”法术的效果——变得灰飞烟灭。

法术效果
受术的不死生物每丧失一点生命值，施法者将受到一点医疗，直至其正常的生命值满值。而若受术目标意志豁免成功，你将失去对它的控制权。


纠缠根须（Entangling Roots）
横生的杂草纠缠住目标，并逐渐紧勒。
变化系
等级: 医师 3
法术成分: 言语、姿势
施法时间: 标准动作
距离: 中距 (100 英尺+l0英尺/每等级)
目标：单个生物
持续时间: 1回合/每等级(解消)
豁免鉴定: 反射，通过则无效
法术抗力: 不可 (见说明)

描述
横生的杂草缠绕着目标的腿部，紧紧地将它勒住。如果目标脚下没有任何植物，则法术会自动制造一些出来。

法术效果
被纠缠的生物将承受攻击检定-2与有效敏捷值-4的惩值。任何被纠缠的角色若意图施法，则必须通过一个DC15的专注检定，否则将失去该法术。被纠缠的生物可使用一个全回合动作来进行一次DC为25的力量检定，或是DC22的脱逃检定。纠缠根须的AC为10，生命值为22。
被缠绕时，受术者将每轮受到1d4点的挤压伤害。


自然之力（Force of Nature）
一棵树转化为树人，并为施法者而战。
变化系
等级: 鹰之德鲁依 6
法术成分: 言语、姿势
施法时间: 1整轮
距离: 接触
目标：接触到的树木
持续时间: 1回合/每等级(解消)
豁免鉴定: 不可
法术抗力: 不可

描述
此法术会将一棵接触到的树木变成一名树人，并在短时间内为施法者而战。

法术效果
自然之力必须被施放在一棵有生命的、健康的树木上。树人的属性见《怪物手册》中的描述。如果自然之力法术被解消，那么不论身处何处，它都会就地生根。而若是施法者自己解除法术，那么树木在扎根前会试图回到原位。


高等自然之力（Force of Nature, Greater）
等级: 鹰之德鲁依8

与自然之力相同，除了效果变为创造出1d4+1名树人。


霜冻护甲（Frost Armor）
给予护甲等级+4加值，并冻结攻击者
咒法系（创造）[力场]
等级: 萨满 2, 术士/法师 2
法术成分: 语言, 姿势, 器材/法器
施法时间: 标准动作
距离: 接触
目标：接触到的生物
持续时间: 1小时/每等级(解消)
豁免鉴定: 意志通过则无效（无害）
法术抗力: 可 (见说明)

描述
一种几乎看不见的力场围绕着“霜冻护甲”所保护的目标。目标的周围将变得十分寒冷。

法术效果
霜冻护甲将给予目标+4护甲加值。与普通的盔甲所不同的是，霜冻护甲不会有盔甲检定减值、奥术施法失败率以及速度减值。也因为霜冻护甲是由力场所形成的，灵体生物无法像穿越普通盔甲那样穿越它。
任何用其肢体或手持武器击中受术目标的生物将造成正常伤害，但同时攻击者将被冻结。生物的法术抗力可以用来对抗冻结效果。要注意的是：有着较大触及距离的武器，例如双手矛等，则不会让使用者受到冻结的威胁。
被冻结的生物的移动以及攻击速度在一轮内都将大幅度减缓。被冻结的生物只能进行移动动作或是标准动作，但不可以在一轮中同时进行这两个动作（也不能进行全回合动作）。此外，他们还将在护甲等级、近战攻击检定、近战伤害检定以及反射豁免上遭受-2减值。被冰冻的生物的跳跃距离只有正常时的一半。

器材/法器:
奥术——一小片锻造过的铁；神术——萨满的图腾

霜之新星（Frost Nova）
爆发出寒冷伤害并且使目标冰冻
塑能系[寒冷]
等级: 萨满 3. 术士/法师3
法术成分: 姿势, 语言, 材料
施法时间: 标准动作
距离: 近距 (25 英尺 + 5 英尺/每2等级)
影响区域: 半径10英尺爆发
持续时间: 立即
豁免鉴定: 见说明
法术抗力: 可

描述
霜之新星是一团带着低鸣的，同时爆发出寒冷伤害和冰冻效果的爆发性冷气团

法术效果
对区域内所有生物造成每施法者等级1D6（最高10D6）的寒冷伤害，反射豁免通过的话，则承受一半伤害，未防护的物品也同样受到伤害，爆炸不会带来风压。
施法者指定欲发动爆炸的爆炸点（距离和高度），既从指尖飞出一个豆粒大小的冰珠，除非是在到达指定点之前撞到物品或者阻碍导致提前爆炸，不然霜之新星会在指定地点爆炸。如果你想冰珠穿过远方的狭缝，必须进行远程接触攻击以“命中”开口，否则冰珠撞到阻碍会提前爆炸。
被霜之新星攻击的生物必须通过坚韧鉴定，否则被冰冻1D4轮。被冰冻的生物移动和攻击都将彻底的变缓慢。被冰冻的生物每轮只能有一个标准动作（但是他们仍然可以进行全回合操作），另外被冰冻的生物在AC，攻击鉴定和伤害鉴定，以及反射鉴定中有-2的惩罚，被冰冻的生物只能跳平时的一半远。

材料成分:
一些雨水和玻璃，必须在施法的时候投出去打碎。


宁静（Healing Rain）
召唤一场大雨，为降雨范围内的所有友军单位恢复生命值。
咒法 (医疗)
等级: 医师 5
法术成分: 姿势, 语言
施法时间: 标准动作
距离: 20 英尺.
影响区域: 以你为中心半径20英尺内全部队友
持续时间: 专注，至多1轮/每等级
豁免鉴定: 坚韧，通过效果减半 (无害)
法术抗力: 可 (无害)

描述
充满正能量的雨水从天而降。

法术效果
在影响范围内治疗的雨水每回合为每个队友治疗1D4伤害。
象其他法术一样,宁静治疗的伤害同时在影响区域给给予不死生物同样的伤害。


治疗守卫（Healing Ward）
用图腾的力量治疗全部范围内的盟友，并且对不死生物造成伤害。
咒法 (医疗)
等级: 萨满 3
法术成分: 姿势, 语言
施法时间: 标准动作
距离: 0 英尺
影响区域: 以图腾为中心的半径20英尺爆发。
持续时间: 1轮/每等级
豁免鉴定: 坚韧，通过效果减半(无害)
法术抗力: 可 (无害)

描述
召唤一根不可思议的图腾插在地面上

法术效果
这个图腾在被插到地上以后将立即开始散发出正能量，它将为范围内所有的盟友每回合恢复1点生命，直到盟友的最大生命值。
象其他法术一样, 治疗守卫治疗的伤害同时在影响区域给给予不死生物同样的伤害。
这个图腾是能够被摧毁的，它的AC是7，硬度是5，HP是5。如果该图腾被摧毁，则法术结束。


献祭（Immolation）
一股包围着施法者，伤害敌人的同时也能保护他免受寒冷伤害的火焰。
塑能[火]
等级: 萨满 4, 术士/法师 4
法术成分: 姿势, 语言, 材料
施法时间: 标准动作
距离: 自己
目标: 自己
持续时间: 1轮/每等级(解消)
豁免鉴定: 不可
法术抗力: 可 (见说明)

描述
施法者的周围环绕着蓝绿相间的火焰。

法术效果
该法术将造成大量热量，并每轮对5英尺内的所有生物造成1d6的火焰伤害。生物的法术抗力对此伤害有效。火焰还将造成与阳光棒同等的照明效果（30英尺）。
这种火焰同样可以保护施法者免受寒冷伤害。角色将只承受寒冷效果的一半伤害。若此种寒冷攻击可以通过豁免减半，那么施法者在成功通过检定后将完全不受伤害。

材料成分:
少量的磷。


闪电盾（Lightning Shield）
一股电流笼罩在施法者身上，并伤害所有临近的生物。
塑能[电]
