职业描述
第一章·灵能职业

　
　　灵能人物开发精神与肉体的力量。
　　　
　　每个有知觉的生物都有生物能源，而生物能源产生意识。“生物能源”有许多面，智力、感知和魅力是意识的直接元素，但力量、敏捷和体质也起作用。每种意识元素都好似白光散射出的颜色之一。灵能职业直接利用意识的全部能量，使用这种内蕴的能力来影响真实世界的改变。普通生物脑中的灰色边界使得思维与梦毫不起眼，但灵能人物的精神意愿却清晰可见。
　　　
　
职业描述
CLASS DESCRIPTION
　
　　本章将详述两个灵能职业：心灵术士和心灵武士。你将会先见到以“游戏世界”内的措辞进行的简单介绍，这是游戏世界里人们的理解方式，也是该人物自我介绍时可能使用的方式。这些描述只是一般性简介，不同个体很可能在心态、观念和其他方面与众不同。
　　　
　　职业简介之後是游戏规则信息。
　　　
　　属性：属性条目说明了哪些属性对于该职业的灵能人物最重要。
　　　
　　阵营：介绍了可能存在的阵营限制。虽然基本灵能职业对任何阵营的人物开放，但进阶职业可能有阵营前提，操焰者(见第六章“人物角色”)就是这样的职业。
　　　
　　生命骰：生命骰类型决定了灵能人物每级获得的生命值多少。灵能人物升级时掷一次生命骰，加上或减去体质调整值，最後把结果加入总生命值。因此，人物的生命骰数和等级相等。1级新人物的第一个生命骰获得最高数值(4或8)而无须掷骰（体质调整值仍适用）。
　　　
　　本职技能：1级灵能人物拥有的技能点数、升级时可以得到的技能点数和本职技能列表。此数值加上或减去智力调整值，就是升级时该人物可获得的总技能点数（即使有智力减值，至少也可获得1点）。记住1级人物的技能点数是此数的四倍。由于本职技能的最高级数是人物等级+3，因此1级人物最高可将本职技能升到级数4，每个级数需耗费1点技能点数。
　　　
　　你也可以把技能点数用在跨职技能上，但每1点技能点数只能得到半个跨职技能级数，而且跨职技能的最高级数是本职技能的一半（因而1级人物的跨职技能最高只能到级数2）。某些灵能本职技能是专职技能，其他职业的人物通常无法学习。
　　　
　　职业特性：该灵能职业的特殊性质。职业特性包括以下项目：
　　　
　　擅长武器与防具：该人物擅长何种武器和防具。
　　　
　　异能：灵能职业使用心灵异能。该条目提供了额外的指导。
　　　
　　其他特性：每个职业都有一些独特的能力。
　　　
　　起始金钱：如果你不是用起始套组，这里注明了人物获得的起始金钱。
　　　
　　起始套组：此处提供1级灵能人物的缺省专长、技能、装备和其他细节。你可以完全不理会起始套组而自创新人物，也可以使用套组作为你的第一个人物，或者混合搭配。地下城主(DM)也可以使用这些套组快速创建1级非玩家人物(NPC)。
　　　
　　起始套组假设你在每个技能上花费4点技能点数（以便专精数门而非博而不精），因而它们的标示方法可能与标准系统略有不同，但其技能级数完全正确。
　　　
　　起始套组按种族分别设计，但并未注明种族特性和属性修正值，故这些须从《玩家手册》中记下。

心灵术士(PSION)
PSION
　
　　强力的拳脚，闪耀的剑光，在心灵术士的一念之间就已经黯然失色。心灵异能源起于长期严格的自我审查和心志锻炼对自身精神能力的培养。那些克服了心魔和种种意想不到的困难能够摒除心中杂念的人们学会了如何唤起体内潜藏的力量。
　　　
　　心灵术士通过不断的研习自身心灵意识来发掘出更为广泛强大的精神力量。他们冥想深思自己的记忆和记忆的本质，与自己的人格片断争辩，并且深入到心中每一个黑暗的角落。“认识自己(know thyself)”不只是心灵术士的格言，更是通向强大力量的道路。
　　　
　　冒险：心灵术士参加冒险只是为了激发自己的心智。全新的体验和经历能带给心灵术士全新的思考方式并最终发掘出潜在的能力。心灵术士的力量是天生的，但并不是轻易就能发挥出来的。
　　　
　　善良的心灵术士能敏锐地意识到不道德地运用心灵异能将会对受术者心灵造成的影响，所以他们只在迫不得已的时候才运用异能攻击一个敌人的心灵并且力求最小量的渗入。然而邪恶的心灵术士对于精神暴力就根本不会如此不安，他们就算污染了那些挡在他们面前的人们的灵魂，也只是把这作为他们自身精神力量优越的证明。
　　　
　　特质：心灵术士从他的脑中提取力量，这种力量是他通过独特的自我研习所获得的学识的逻辑表达。心灵术士拥有的异能比同级的法师或术士的法术要少，但是他非常喜爱异能所具有的极高的灵活性。心灵术士最初须选择一个基本能系，从而获得附加的心灵能力，尽管他在相关属性值足够高时可以学习任何能系的异能。
　　　
　　{ Andor注：学习（发掘）异能并无属性要求，只在展现时须显能者的该异能所属能系关键属性值足够。 }
　
　　心灵术士可以在一枚水晶或宝石内注入自己的人格片断来创造一个灵晶仆(Psicrystal，类似魔宠)。灵晶仆拥有对心灵术士很有帮助的特殊能力。
　　　
　　心灵术士自身知识增长时便会获得新的异能。但是为了真正地认识自己，心灵术士不只需要内心的修行，同时也需要体格上的，所以他们通常会学习一点基本的战斗技巧并擅长简易武器。
　　　
　　阵营：对于心灵术士来说，灵能只是个人技艺。心灵术士们看起来都比较内向，所以他们稍稍倾向于中立阵营而非混乱或守序，但并非必须中立。均衡来看，心灵术士中既有善良的也有邪恶的。
　　　
　　宗教：由于特别关注他们自身的潜能，心灵术士们并未显著地信仰任何神祗。崇敬神祗的心灵术士们并无规律可循。(Psions who do revere a deity do not conform to any general choice.)
　　　
　　背景：注定要成为心灵术士的人们在幼年时就表现出精神力量的征兆，到了青春期这种表现变得特别显著，像出现莫名其妙的声音和光芒，陶器自己在空中乱飞，小物件无中生有地出现等都是很常见的。大部分拥有这种天赋的人都没有发掘自己的潜能，而另一些欣然接受了这种特殊能力。从那之後，崭露头角的心灵术士开始练习并改善他们的能力。如同舞弄法术的术士一样，因为朋友的畏惧和家人的误解，心灵术士们通常都很孤僻。有时，心灵术士也能找到其他有类似能力的伙伴并组成非正式团体甚至是聚集拥有心灵能力者的小型社区。
　　　
　　种族：灵能这种天赋不可预测，在任何普通种族中都有可能出现心灵术士。
　　
　
　　灵吸怪(illithids)非常可能成为心灵术士，因为他们的能力实际就是深层灵能本质的成分，他们甚至不须心灵术士等级就已拥有可怕的心灵力量。
　　
　
　　与其他职业的关系：心灵术士们欣赏与他们同样投身于自我控制的武僧团体。有些心灵术士看不起法师、术士这些施法者，因为他们觉得魔法只是心灵的拐杖，精神不完善的人才需要魔法。心灵术士常常和心灵武士互相戏弄，但他们的对抗很少导致彻底的冲突。
　　　
　　心灵术士的能力使他对任何冒险团队来说都是一个强有力的伙伴。不同能系的心灵术士在队伍中扮演着不同的角色。例如，心灵预言(Clairsentience)专家通过高难度侦察引领队伍，而选择心灵传送系 (Psychoportation)作为基本能系的心灵术士则能大大提高队伍的机动力。

游戏规则信息
GAME RULE INFORMATION
　
　　心灵术士拥有以下游戏数据。
　　
　
　　属性：每一项属性对心灵术士来说都可能至关重要，因为每一能系都有影响该能系异能使用的关键属性(见关于能系的边栏)。每一异能都属于六大能系之一，每一能系都有一项不同的关键属性。例如，敏捷是心灵传送系的关键属性。特定能系内的所有异能受同一项属性影响。
　　　
　　当心灵术士使用心灵异能时，称为“展现”，“展现异能”简称“显能”。欲展现任何异能，心灵术士的关键属性值必须达到10加该异能能级（就是说，某能系的关键属性值小于10的心灵术士无法展现该能系的异能）。心灵术士还必须拥有足够的灵能点支付异能费用。
　　　
　　选择属性值很重要的另一个原因是防御者对抗心灵术士攻击者的异能的意志豁免DC是由攻击者每次显能时决定的：1d20 + 该异能能级 + 关键属性调整值(详见第四章“灵能综述”)。
　　　
　　心灵术士还根据他的基本能系的关键属性值获得的额外灵能点（这点从不改变）。表1-2“心灵术士的额外灵能点”显示了基本能系拥有高属性值的心灵术士升级时获得的额外灵能点数。这些点数是加成的；因此，关键属性值18的5级心灵术士有 1 + 3 + 5 = 9 点额外灵能点。而关键属性值为15的8级心灵术士只有 1 + 3 = 4 点额外灵能点，和他较低等级时一样。额外灵能点只由24小时无波动的高属性值产生。
　　　
　　除了高属性值，心灵术士必须有足够等级才能获得该等级的额外灵能点。例如，心灵术士亚历珊卓（她的基本能系为心灵传送系）在3级时敏捷为16，足以在1级获得1点额外灵能点并在3级获得3点额外灵能点，总共4点。她达到5级时才能再获得5点，也就是总共9点。
　　　
　　如果你的人物关键属性值为9或更低，则无法展现出该属性相关的（也就是对应能系的）心灵异能。例如，若亚历珊卓的敏捷因中毒或心灵战斗而降为9，那么在被治愈前她都无法再展现任何即使最简单的心灵传送系异能。
　　　
　　阵营：无限制。
　　　
　　生命骰：d4。

能系
DISCIPLINES
　
　　心灵异能分为六大能系(Disciplines)，每系都有共同主题，如心灵创造系和心灵附魔系。每一道已知异能都属于六大能系之一。在创建人物的时候，心灵术士必须选择一个能系作为基本能系，基本能系一旦选定就不能改变。人物可以依据表1-1选择任何能系的心灵异能并且每异能能级还将知道基本能系的 1个额外异能。基本能系还决定了由哪项关键属性提供心灵术士额外灵能点(见表1-2)。

　　心灵传送系(Psychoportation)：能使人物或物品时空旅行的心灵异能。选择心灵传送系作为基本能系的心灵术士被称为灵旅者(Nomad)。敏捷是该能系的关键属性，并决定灵旅者的额外灵能点数。

　　心灵创造系(Metacreativity)：能通过星界摄取物质，创造实体或半实体物品的心灵异能。选择心灵创造系作为基本能系的心灵术士被称为创形者(Shaper)。智力是该能系的关键属性，并决定创形者的额外灵能点数。
　　　
　　心灵预言系(Clairsentience)：揭示隐藏信息的心灵异能。选择心灵预言系作为基本能系的心灵术士被称为洞察者(Seer)。感知是该能系的关键属性，并决定洞察者的额外灵能点数。
　　　
　　心灵附魔系(Telepathy)：能够同其他有意识的生物建立精神联系或进行心灵控制的心灵异能。选择心灵附魔系作为基本能系的心灵术士被称为传心者(Telepath)。魅力是该能系的关键属性，并决定传心者的额外灵能点数。

职业特性
Class Features
　
　　以下均为心灵术士的职业特性。
　
　
　　擅长武器与防具：心灵术士擅长所有简易武器，他们不擅长任何盔甲与盾牌。防具不会干扰心灵异能的使用，但当穿戴不擅长的盾牌或盔甲时，攻击检定和以下技能会遭受防具检定减值：“平衡”、“攀爬”、“脱逃”、“躲藏”、“跳跃”、“潜行”、“扒窃”和“翻滚”(关于防具检定减值见《玩家手册》第七章)。{ Andor注：穿戴擅长防具时所列技能也同样有减值。 }
　　　
　　异能：心灵术士展现异能。心灵术士能选择的异能非常有限，然而享有极大的灵活性。1级的心灵术士只知道所选的2个零级异能（或称为“天赋异能”）和0个一级异能再加上基本能系的每能级1个额外异能，一共3个零级异能和1个一级异能。心灵术士可随着升级发掘出一个或多个潜藏的异能，如同表1-1“心灵术士”所标明的，心灵术士可从任何能系选择异能，但表1-1每列中的“d”表示基本能系的额外异能。“--”表示并无已知异能，“d”或数值表示心灵术士在该级可以选择若干异能。
　　　
　　已发掘的异能可以是心灵术士异能列表中的一般异能 (见第五章“异能详述”)或心灵术士通过研习领悟到的特殊异能。例如，一个心灵术士有一枚注有（心灵术士异能列表上所没有的）特殊异能的灵能石，他便可以在升级获得新异能时选择这道异能，当然他必须达到展现这道异能的等级。请注意，在任何情况下心灵术士都不可以用这种方法习得超过“表1-1”所规定数量的心灵异能。
　　　
　　欲展现某一能系的异能，心灵术士的关键属性值至少要达到10 + 异能能级。（心灵攻击和防御模式不受这个限制，也不具有能级。）抵抗心灵术士异能的豁免检定DC为1d20 + 异能能级 + 心灵术士的关键属性调整值，该1d20由心灵术士攻击者在展现异能时掷骰。(心灵攻击和防御模式使用特定修正值，详见第四章。)
　　　
　　另外，在展现某些异能时可以支付更多灵能点以增强效果。第三章“专长”这部分详述了很多增强心灵异能的专长。
　　　
　　注意：一名心灵术士在升级时可以选择发掘较低能级的异能来代替正常能级的异能。因此，刚从8级升到9级的心灵术士可以选择习得一个三级或更低能级的异能来代替四级的。
　　　
　　灵能点：心灵术士每日可展现的异能数量由他的可用灵能点数决定，但他不用预先准备异能——他只要在展现异能时支付相应的灵能点费用。表1-2中显示的额外灵能点可增加心灵术士的每日可用灵能点数。例如，传心者艾达波迪在1级时可得到2点每日灵能点数，加上足够的魅力（他的基本能系的关键属性）带来的1点额外灵能点，他还选择了“内心强健”专长，所以还可得到1点灵能点，也就是一共4点每日灵能点数。
　　　
　　零级异能：又称“天赋异能(talents)”。对于天赋异能，有特殊的灵能点花费规则。心灵术士每日可免费任意展现 职业等级 + 3 道天赋异能，用尽了这个次数以後，该日内每展现一道零级异能需花费1点灵能点。因此，一个1级心灵术士每日可免费展现4道天赋异能，此後若在同一天内还要展现零级异能则每道需花费1点灵能点。
　　　
　　心灵战斗模式：1 级时，心灵术士可以选择习得10项心灵战斗模式中的5项。(心灵战斗模式的具体描述和设定见第四章“灵能综述”。)此後，在3级、5级、7级、9级和11 级时又可分别再获得1个心灵战斗模式(如表1-1所示)。五项攻击模式为“自性鞭击”、“本我暗示”、“心灵爆破”、“心灵穿刺”和“心力榨取”。五项防御模式为“心灵空白”、“智力堡垒”、“精神障壁”、“思维之盾”和“坚固意志”。
　　　
　　心灵攻击和防御模式在对付灵能人物和生物时特别有效，但也可影响非灵能生物。在某些方面心灵战斗模式很像异能，但它们具有异能所没有的特殊性质。
　　　
　　心灵战斗模式有不同的展现费用，这里提醒低级心灵术士选择低费用的战斗模式。例如，1级心灵术士可以选择“心灵爆破”，但高达9的灵能点费用使其在近几级内都难以使用。
　　　
　　灵晶仆：心灵术士能够创造一枚灵晶仆。创造灵晶仆的过程需要花掉一天，还需价值100金币的材料（通常作为灵晶仆基体的水晶或宝石至少价值总花费的50%）。灵晶仆形如一枚直径不到1吋的透明水晶石，其载有心灵术士自己的人格片断，因而与心灵术士有精神联系，可以增强心灵术士的部分能力。一名心灵术士同时最多只能拥有一枚灵晶仆。
　　　
　　心灵术士在创造灵晶仆时选择灵晶仆的种类，就是说，决定将自身哪一部分人格片断注入水晶石。当心灵术士升级时，灵晶仆的能力也随之提升，见表1-5“灵晶仆的特殊能力”。
　　　
　　灵晶仆被摧毁时，心灵术士必须进行强韧检定(DC=15)，若未通过则失去每等级200点经验值，通过则减半。但是心灵术士的经验值不会因此而减到0以下。例如，当艾达波迪的灵晶仆被索尔石怪吃掉时，他是有着3230点经验值的3级心灵术士。艾达波迪通过了豁免，因此将失去300点经验值，使他的经验值低于3000而掉回2级(《地下城主指南》中有关于失去等级的规则)。一旦灵晶仆被摧毁，心灵术士必须等到六个月後才能再次创造新的灵晶仆。
　　　
　　起始金钱：3d4×10金币（除非使用起始套组）。
　　Starting Gold: 3d4x10 gp (unless using a starting package).

　　灵晶仆人格描述：所有灵晶仆都有独特的人格。心灵术士在1级时只能强烈感觉到灵晶仆的人格，但在高级时灵晶仆可以自由地通过心灵感应向心灵术士提供建议，这些建议强烈倾向于灵晶仆自己的世界观。主人总是能从他的灵晶仆中看见一部分自己，即使是由于夸大而扭曲的自己。
　　　
　　巧匠(Artiste)：这小片人格毫不谦虚地表达自己在几乎所有职业上有惊人的天赋，例如舞蹈、歌剧和烹饪。同样多的誓言和自负充满了该宣言。
　　　
　　恶霸(Bully)：有时坏品格对某些事而言是倒是优点。恶霸人格很讨厌，它蛮横而夸夸其谈并且非常固执地认为自己是对的，但心灵术士可以不时善用这种影响。
　　　
　　懦夫(Coward)：健全的自卫本能深藏在每个人的心底。当懦弱的要素被捕捉并放大时，可以帮助心灵术士在危险情形下寻找安全港。然而，这种灵晶仆过分热衷于催促心灵术士逃跑并躲藏，即使只是在一般危险的情形下。
　　　
　　友善(Friendly)：这小片人格乐于助人，甚至是敌人。心灵术士主人得忍受那针对他的硬心肠的说教，结果就是心灵术士会更乐于以非暴力方式解决对手。
　　　
　　英雄(Hero)：我们都想成为英雄，灌输了这部分人格的灵晶仆便是这种感觉的人性化。它惯于激励心灵术士进入危险境遇，并回报给主人以在任何情形下的强韧身体。
　　　
　　惯谎(Liar)：爱说谎的灵晶仆会情不自禁地说谎，即使是对深谙它何时在说谎而何时在说实话的主人。当心灵术士打算说谎的时候它的说谎方式就能帮上忙了。
　　　
　　细心(Meticulous)：细心的灵晶仆花费大量时间辨别重要细节和无关紧要的细节。它对自己的描述是“对礼节一丝不苟，而且极端注重细节”。这种特点虽然有时几乎令人发狂，但对于试图仔细研究所处环境的心灵术士来说是很有用的。
　　　
　　顾盼(Observant)：“你注意那个了么？”是灵晶仆最常告诉主人的。心灵术士常常早已看见灵晶仆提示的东西并由于不重要而忽略了。然而有时这种特点会提醒心灵术士注意到忽略了的东西。
　　　
　　沉着(Poised)：时时注意自己的身体和环境的配合常常会导致失去更重要的想法。Not so for this psicrystal, which is perfectly poised and sure of itself, if not its owner. 它总是宣扬平衡感的重要性并随时准备行动，它的影响倒确实能在平衡比较重要时给予心灵术士帮助。
　　　
　　坚毅(Resolve)：该灵晶仆正拥有这种所有人都非常想要的品质。它的意志和信仰非常坚固，坚定和决心体现在它的建议和意见中。或者，用心灵术士的话说，他是这样顽固。然而，这种坚定不移的信念帮助心灵术士进行一切意志对抗。
　　　
　　智者(Sage)：这小片人格试图成为主人的良师，即使它本身就是从主人的精神中抽出的。然而，脱离了其他人格类型，智者认为自己是精神和哲学上最重要的大师，而不提及自己专精的知识领域。智者灵晶仆超出专业以外的建议常常是不可靠的。
　　　
　　专心(Singleminded)：植入该灵晶仆的是忽略环境和情感因素而专注手头工作的能力。有时心灵术士很难引起专注在某件特定任务上的灵晶仆的注意，但它的影响也扩大了心灵术士自己的专注能力。
　　　
　　静谧(Sneak)：可以用比呼吸声还轻的脚步声出现时为何要用沉重的脚步声引人注意呢？全身心信仰沉默是金的作用的静谧灵晶仆大概会这么问。它惯于使主人和主人的同伴高度安静。然而，它在安静移动方面的专业性不可否认。
　　　
　　同感(Sympathetic)：这种性质使灵晶仆设身处地地看穿其他生物的心思。不幸的是，灵晶仆花大量时间分析心灵术士及其同伴，评论他们的心理状态，并且对他们行动的适当性发表判断，这种能力在心灵术士试图确定潜在敌人的动机时非常有用。
　　　
　
　　心灵术士等级：心灵术士主人的等级。

心灵武士(PSYCHIC WARRIOR)

 心灵武士(PSYCHIC WARRIOR)

　
　　心灵术士通常全力挖掘自己内心深处的力量，而另一部分灵能者则更专注于锻炼肉体，这些蕴含着奇异精神力量的武士通常被称为心灵武士，他们内外兼修，同时运用自己的头脑和肌肉。
　　　
　　阔剑，战斧，长弓，这些都是用来展现灵能力量的工具。同时修炼体格和灵能的心灵武士是值得人们敬畏的对手。
　　　
　　冒险：心灵武士都知道，只有在冒险中才能锻炼自己的力量，同时也只有冒险才能带来财富和更好的装备。靠着那令人激赏的力量，他们无所畏惧。
　　　
　　特质：心灵武士最大的特点就是利用灵能专长和心灵异能强化自己的物理攻击。结合了肌肉、战斗技巧和心灵异能的心灵武士拥有足以匹敌，甚至超越普通战士的力量。
　　　
　　当心灵武士在冒险中成长，他的战斗水准和灵能力量会同时提升。
　　　
　　阵营：一名心灵武士需要同时接受心灵和肉体这两种截然不同的训练，因此通常属于混乱阵营，但并非必要条件。
　　　
　　宗教：心灵武士倾向于非守序阵营的神祗，常常选择勇者之神海若尼斯、力量之神寇德或是杀戮之神厄瑞斯努。但与心灵术士相似，心灵武士也更相信自己内心深处的力量，所以通常对神祗不怎么感兴趣。
　　　
　　背景：一般而言，心灵武士在心灵术士组建的“同志会”团体或小型社区中成长。这种社区非常罕见，但它们的存在为常常受到世人敌视的拥有纯粹心灵能力的人提供了一些保护。心灵武士通常是心灵术士的孩子，他们在成长中不愿完全遵从父母的意愿。缺乏灵能高手指导的外人很难成为心灵武士。
　　
　
　　虽然他们的职业离经叛道，但大部分心灵武士仍感到和养育自己的社区或团体有着密切关系。当然，也有一些心灵武士只关注他们自己的个人修炼之路。
　　　
　　种族：灵能这种天赋不可预测，在任何普通种族中都有可能出现心灵武士。
　　　
　　夺心魔更容易成为心灵术士，而不是有着双重训练这种混乱观点的心灵武士。然而确有少量的灵吸怪心灵武士存在。
　　　
　　与其他职业的关系：不管是什么职业，只要尊重和欣赏心灵武士的独特力量，任何人都可以和心灵武士融洽相处。又由于心灵武士善于战斗，所以基本上任何冒险者都会欢迎他们的加入。

职业特性
Class Features
　
　　以下均为心灵武士的职业特性。
　　

　　擅长武器与防具：心灵武士擅长使用所有简单武器和军用武器，以及所有的盔甲（轻型、中型、重型）和盾牌。请注意，当穿戴盾牌或比皮甲重的盔甲时，以下技能会有防具检定减值： “平衡”、“攀爬”、“逃脱”、“躲藏”、“跳跃”、“潜行”、“扒窃”、“翻滚”。此外，“游泳”技能检定时，身上每带有5磅的防具与装备，就会累计- 1减值。
　　　
　　额外专长：1级时，心灵武士可以得到一个额外专长（这是加在1级人物都有的一个专长以及人类特有的一个额外专长之外的）。2级时，可再获得一个额外专长，之後每提升3级时（5级、8级、11级，依此类推），都可获得一个额外专长。这些额外专长只能从灵能专长和《玩家手册》所载的以下专长中选择：“双巧手”、“盲斗”、“战斗反射”、“闪避”（“灵活移动”、 “跳跃攻击”）、“擅长异种武器”*、“寓守于攻”（“精通卸除武器”、“精通绊摔”、“旋风攻击”）、“精通重击”*、“精通先攻”、“精通徒手击打” （“拨挡飞箭”、“震慑拳”）、“骑乘战斗”（“骑乘射击”、“踩踏”、“快速骑乘攻击”、“奋力冲刺”）、“近程射击”（“远程射击”、“精准射击”、 “快速射击”、“移动射击”）、“猛力攻击”（“顺势斩”、“精通冲撞”、“精通击破武器”、“强力顺势斩”）、“即时备战”、“双武器攻击”（“精通双武器攻击”）、“武器娴熟”*和“专攻武器”*。
　　　
　　上述某些专长须先获得某些作为先决条件的专长，否则不能选择，许多灵能专长同样有此限制，详见本书第三章。上述名称後以*号标示者，表示可以选择多次，但每次必须选择不同武器。选择专长时，必须满足所有先决条件，包括属性值和最低基本攻击加值（见《玩家手册》第五章各专长描述）。
　　　
　　重要说明：心灵武士每三级仍可获得一个专长，而且不受上述专长列表限制。
　　　
　　异能：心灵武士展现异能，但能选择的范围比心灵术士更为有限。1级的心灵武士只知道2个零级异能（天赋异能）。心灵武士可随着升级发掘更多潜藏的异能，如同表1-6“心灵武士”所标明的，心灵武士所知道的异能必须从心灵武士异能列表中选择（见第五章“异能详述”）。
　　　
　　欲展现某一能系的异能，心灵武士的关键属性值至少要达到10 + 异能能级。（心灵攻击和防御模式不受这个限制，也不具有能级。）抵抗心灵武士异能的豁免检定DC为1d20 + 异能能级 + 心灵武士的关键属性调整值，该1d20由心灵武士攻击者在展现异能时掷骰。（心灵攻击和防御模式使用特定修正值，详见第四章。）
　　　
　　另外，在展现某些异能时可以支付更多灵能点以增强效果。第三章“专长”部分详述了很多增强心灵异能的专长。
　　　
　　注意：一名心灵武士在升级时可以如同心灵术士那样选择发掘较低能级的异能来代替正常能级的异能。
　　　
　　灵能点：心灵武士每日可展现的异能数量由他的可用灵能点数决定，但他不用预先准备异能——他只要在展现异能时支付相应的灵能点费用。心灵武士不能通过属性值获得额外灵能点，因此一名1级的心灵武士只有2点灵能点。心灵攻击模式，零级异能和心灵战斗模式遵循特殊规则。
　　　
　　零级异能：又称“天赋异能”。对于天赋异能，有特殊的灵能点花费规则。心灵武士每日可免费任意展现 职业等级 + 2 道天赋异能，用尽了这个次数以後，该日内每展现一道零级异能需花费1点灵能点。因此，一个1级心灵武士每日可免费展现3道天赋异能，此後若在同一天内还要展现零级异能则每道需花费1点灵能点。
　　　
　　心灵战斗模式：1级时，心灵武士可以选择习得10 项心灵战斗模式中的2项。（心灵战斗模式的具体描述和设定见第四章“灵能综述”。）此後，心灵武士还可以按照表1-6“心灵武士”再获得心灵战斗模式。五项攻击模式为“自性鞭击”、“本我暗示”、“心灵爆破”、“心灵穿刺”和“心力榨取”。五项防御模式为“心灵空白”、“智力堡垒”、“精神障壁”、“思维之盾”和“坚固意志”。
　　　
　　心灵攻击和防御模式在对付灵能人物和生物时特别有效，但也可影响非灵能生物。在某些方面心灵战斗模式很像异能，但它们具有异能所没有的特殊性质。
　　　
　　心灵战斗模式有不同的展现费用，这里提醒低级心灵武士选择低费用的战斗模式。例如，1级心灵武士可以选择“心灵爆破”，但高达9的灵能点费用使其在近几级内都难以使用。
　　　
　　武器专精：心灵武士达到6级时自动获得“武器专精”专长。武器专精可以让特定武器具有+2伤害加值。心灵武士必须先具有该武器的“专攻武器”专长，才能够选择该武器的“武器专精”。如果该武器是远程武器，则只有当目标位于30呎内，才具有伤害加值，因为这样才可能准确击中敌人要害。心灵武士是作为免费专长获得“武器专精”的，不计入人物的额外或常规专长。人物不能使用额外或常规专长学习“武器专精”。
　　
　
　　起始金钱：5d4×10金币（除非使用起始套组）。

兼职灵能人物

　
　　灵能人物在升级时可以选择新的职业。兼职可以给人物以更广泛的能力，但牺牲了灵能职业的深度。例如，心灵术士艾达波迪可以成为心灵术士/战士。新的战士职业使他擅长更多的武器，拥有较好的强韧豁免等等，但这也意味着他未能获得新的心灵术士能力并因而不如纯心灵术士强。
　　　
　　注意：心灵术士不可以兼职其他能系心灵术士，但心灵术士/心灵武士兼职是允许的。各职业的灵能点合并计算，但发现的异能须分别记录，因为职业等级影响具体异能的展现。
　　　
　　如果DM允许，其他职业人物可以在升级时选择心灵术士或心灵武士。当非灵能人物拥有了灵能职业，便认为该人物的“野性天赋”觉醒了。
　　　
　　例如，矮人战士托达克升至第5级时可以选取1级心灵武士职业。由于战士是矮人的天赋职业，因此升级其他职业（这里指心灵武士）时，战士职业在计算兼值经验值减值时不计。如果DM允许，人类和半精灵可以选择心灵术士或心灵武士作为天赋职业。某些怪物（包括灵吸怪、吉斯彦克人、吉斯泽莱人）也可以选择灵能职业为天赋职业。（见第八章“怪物”）
　　　
　　关于兼职的全面规则可以参阅《玩家手册》的第三章。
