【STARWARS】d20版一般技能
 

Skill：内容


估价 Appraise Int：

检定普通或常见的物品，误差价格10%内（DC 12）
稀有物品70%～130%，丢2d4+5×10%决定（DC 15）
太空航行 Astrogate Int 计算超空间跳跃路线。耗时10 round，（DC 10）
耗时每减少一轮DC +2。S：使用导航电脑，依电脑优劣可得+2. +4. +6修正，
航太机器人可修正+1. +2. +3。


哄骗 Bluff Cha ：

就是唬烂啦。检定取决于对方的动机查觉技能，并由GM视情况增减。


攀爬 Climb：

Str DC视情况而定。检定失败达5+，摔落！
攀爬中遭到攻击，没有防御Dex修正，对手攻击+2。

电脑使用 Computer Use Int：

变更目前程式（DC 15），新程式（DC 20～30）
破坏安全系统（DC 25）


手艺 Craft Int：

制造物品。DC视物品复杂度而定。
以手艺工作一周，可得技能×20 Credits。


爆破 Demolitions：

Int 设置爆破装置（DC 10），解除爆破装置（DC 15）
制造临时炸弹（DC 25）。


外交 Diplomacy：

Cha 交涉能力。S：Bluff或Sense Motive 5+获得+2修正。

破坏装置 Disable Device Int：

简单的，耗时1 round（DC 10），
微妙的，耗时1d4 round（DC 15），
困难的，耗时2d4 round（DC 20），
恶毒的，耗时2d4 round（DC 20～40）


易容术 Disguise：

Cha DC视情况而定。


娱乐 Entertain Cha：

表演技能。可以赚钱哟！DC与收入视表演难度而定。


脱逃术 Escape Artist：

Dex 绳索（DC 视对手Dex检定+10），网子（DC 20），
眼罩与手铐（DC 30～35），紧闭空间（DC 30），
格斗者（DC 视对手grspple检定）


伪造 Forgery：

Int DC视内容而定。


探听情报 Gather Information：

Cha DC视情况而定。


控制动物 Handle Animal：

Cha DC与消耗时间视情况而定。


隐蔽 Hide：

Dex DC视对手的侦察技能而定。移动只有一半的速度，以全速移动检定-5。另有体型修正。


胁迫 Intimidate：

Cha DC一般为对手等级+10。


跳跃 Jump：

Str 奔跑跳远，基本2m，额外 检定超过10每1点+ 1/3m，上限 身高×6立定跳远，基本1m，额外 检定超过10每2点+ 1/3m，上限 身高×2奔跑跳高，基本1m，额外 检定超过10每4点+ 1/3m，上限 身高×1.5立定跳高，基本1m，额外 检定超过10每8，基本1/3m，，额外 检定超过10每8点+ 1/3m，上限 身高

知识 Knowledge：

Int


倾听 Listen：

Wis DC视情况而定。


潜行 Move Silently：

Dex DC是对手的倾听技能而定。移动只有一半的速度，以全速移动检定-5，奔跑或冲锋-20。


驾驶能力 Pilot：

Dex DC视载具机动性而定。


专业Profession：Wis


语言读写 Read/Write Language：

Int Lv 1st时，可以读写自己的母语。


维修 Repair：

Int DC视器械复杂度而定。没有适合工具检定-5，外星或是不熟悉的器械DC+5。S：专业和手艺技能5+，获得+2修正，使用电脑技能5+，维修电脑类+2。

骑术 Ride：

Dex DC视动作而定。


搜索 Search：

Int DC视对手隐蔽技能而定。
区域内搜索特定物品（DC 10），发现密门、陷阱和线索（DC 20）

动机查觉 Sense Motive：

Wis 察觉对方是否在欺骗你。DC视对方的哄骗技能而定

妙手空空 Sleight of Hand：

Dex 偷东西或是将东西塞给别人。DC视情况而定。
S：哄骗技能5+，获得+2修正。


额外语言 Speak Language：

Int 限制 语言读写技能1+


侦察 Spot：

Wis DC视对手隐蔽或易容术技能而定。
每4m的距离检定-1，注意力被分散-5。


生存 Survival：

Wis DC视环境状况而定。


游泳 Swim：

Str DC视环境状况而定。检定失败达5+，溺水下沈！
每小时重新检定，失败1d6伤害。
S：每2kg道具重量，检定-1。


伤害治疗 Treat Injury：

Wis 急救（DC 15），长期照顾（DC 15），治疗中毒（DC 视毒素而定），治疗疾病（DC 视疾病而定），使用治疗槽（DC 15）
额外恢复 5-9 1点WP 10-14 2点WP
15-19 3点WP 20-24 4点WP 25+ 5点WP


翻滚 Tumble：

Dex DC视动作而定。翻滚5+能够获得额外的闪躲修正。
DC视情况而定，通常是10～20之间。
