书卷与血脉－新奥术列表
 

----零级奥术(戏法)----
转化系 - 修复微小损伤 - "恢复"1点构造物所受的伤害

----壹级奥术----
塑能系 - 次级酸液球 - 远程接触攻击，造成1d8点酸伤害；每2等级再多发出1个球(最大5个)
- 次级冻气球 - 远程接触攻击，造成1d8点冷伤害；每2等级再多发出1个球(最大5个)
- 次级电击球 - 远程接触攻击，造成1d8点电伤害；每2等级再多发出1个球(最大5个)
- 次级火焰球 - 远程接触攻击，造成1d8点火伤害；每2等级再多发出1个球(最大5个)
- 次级音爆球 - 远程接触攻击，造成1d8点声伤害；每2等级再多发出1个球(最大5个)
死灵系 - 负能量射线 - 远程接触攻击，造成1d6点负能量伤害；每2等级再+1d6(最大5d6)
转化系 - 修复轻微损伤 - "恢复"1d8+1/等级(最大+5)点构造物所受的伤害

----贰级奥术----
防护系 - 过滤术 - 目标获得抵抗毒气的能力
- 抗凝视幕 - 目标有50%的机率躲过凝视攻击
咒法系 - 窒息术 - 无形之手锁住敌人咽喉，每回合造成1d4点伤害
- 冰刃术 - 远程接触攻击，造成1d8点冷伤害和2点敏捷伤害
附魔系 - 静心术 - 目标获得对恐惧、强迫以及影响士气效果的防护
幻术系 - 亡者装 - 外貌变成不死生物
死灵系 - 命令亡者 - 使一个不死生物不攻击你并服从你的命令
- 虚假生命 - 获得1d10+1/等级(最大+10)的临时生命
转化系 - 鹰之风采 - 目标获得1d4+1点魅力，持续1小时/等级
- 狐之机巧 - 目标获得1d4+1点智力，持续1小时/等级
- 枭之灵感 - 目标获得1d4+1点感知，持续1小时/等级
- 修复中等损伤 - "恢复"2d8+1/等级(最大+10)点构造物所受的伤害
通用 - 魔宠袋 - 创造一个隐型的特殊空间放置你的魔宠

----叁级奥术----
咒法系 - 幽魂烛 - 无形的持烛之手为你揭示隐形的生物和物体
预言系 - 奥法视觉 - 你可以看到魔法辉光
塑能系 - 冰爆术 - 大型冰块爆发，30尺半径内造成1d4点冰伤害和1点钝击伤害/等级(最大10d4+10)
死灵系 - 假死术 - 使活的生物变成像死了一样
- 负能量爆 - 20尺半径内造成1d8+1/等级(最大+10)点负能量伤害
转化系 - 修复严重损伤 - 恢复"3d8+1/等级(最大+15)点构造物所受的伤害
通用 - 魔宠增强 - 你魔宠的豁免、攻击骰和AC获得+2修正，持续1小时/等级

----肆级奥术----
防护系 - 群体元素抵抗 - 类似抵抗元素伤害术，每等级影响一个生物
- 欧提路克驱魔幕 - 创造一面驱魔屏障
塑能系 - 酸液球 - 远程接触攻击，造成1d6/等级(最大15d6)点酸伤害
- 冻气球 - 远程接触攻击，造成1d6/等级(最大15d6)点冷伤害
- 电击球 - 远程接触攻击，造成1d6/等级(最大15d6)点电伤害
- 火焰球 - 远程接触攻击，造成1d6/等级(最大15d6)点火伤害
- 音爆球 - 远程接触攻击，造成1d6/等级(最大15d6)点声伤害
死灵系 - 负能量波动 - 澎湃的负能量波动斥责或强化1d6HD/等级(最大15d6)的不死生物
转化系 - 群体暗视术 - 类似暗视术，但是影响10尺半径内的所有生物
- 修复致命损伤 - 恢复"4d8+1/等级(最大+20)点构造物所受的伤害
通用 - 魔宠强化 - 你的魔宠获得+2自然装甲，并且有25%的机率躲过致命一击，持续1小时/等级

----伍级奥术----
防护系 - 能量缓冲 - 吸收1d6/等级(最大15d6)点你指定的一种元素伤害
- 摩登凯宁密室术 - 24小时内阻止任何人窥视或占卜你所在的区域
死灵系 - 幽魂壁 - 创造一道幽魂组成的墙壁，触及者受到1d10点伤害并被能量吸取
转化系 - 鬼化术 - 你变成灵体

----陆级奥术----
死灵系 - 永亡术 - 摧毁1d4HD/等级(最大20d4)的不死生物
转化系 - 群体飞行术 - 类似飞行术，不过每等级多影响1个射程内的生物
通用 - 魔宠驻法 - 你传输法术到魔宠

----柒级奥术----
防护系 - 能量免疫 - 对一种你指定的能量伤害免疫
- 欧提路克强力驱魔幕 - 创造一面驱魔屏障(类似强力驱除魔法)
转化系 - 群体传送术 - 类似传送术，但是可以传送更多物体

----捌级奥术----
防护系 - 空间锁 - 阻挡传送或其他空间旅行奥术，持续1天/等级
塑能系 - 强力咆哮术 - 破坏性的喊声造成20d6点声伤害，并震晕生物和伤害物体

----玖级奥术----
塑能系 - 链式触发 - 类似触发术，但你可以设置最多3个法术
死灵系 - 隐命术 - 你将生命原力封锁在一个小型容器中，只要容器未破损，你就无法被杀死
