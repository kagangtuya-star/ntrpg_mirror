心灵术士(PSION)
PSION
　
　　强力的拳脚，闪耀的剑光，在心灵术士的一念之间就已经黯然失色。心灵异能源起于长期严格的自我审查和心志锻炼对自身精神能力的培养。那些克服了心魔和种种意想不到的困难能够摒除心中杂念的人们学会了如何唤起体内潜藏的力量。
　　　
　　心灵术士通过不断的研习自身心灵意识来发掘出更为广泛强大的精神力量。他们冥想深思自己的记忆和记忆的本质，与自己的人格片断争辩，并且深入到心中每一个黑暗的角落。“认识自己(know thyself)”不只是心灵术士的格言，更是通向强大力量的道路。
　　　
　　冒险：心灵术士参加冒险只是为了激发自己的心智。全新的体验和经历能带给心灵术士全新的思考方式并最终发掘出潜在的能力。心灵术士的力量是天生的，但并不是轻易就能发挥出来的。
　　　
　　善良的心灵术士能敏锐地意识到不道德地运用心灵异能将会对受术者心灵造成的影响，所以他们只在迫不得已的时候才运用异能攻击一个敌人的心灵并且力求最小量的渗入。然而邪恶的心灵术士对于精神暴力就根本不会如此不安，他们就算污染了那些挡在他们面前的人们的灵魂，也只是把这作为他们自身精神力量优越的证明。
　　　
　　特质：心灵术士从他的脑中提取力量，这种力量是他通过独特的自我研习所获得的学识的逻辑表达。心灵术士拥有的异能比同级的法师或术士的法术要少，但是他非常喜爱异能所具有的极高的灵活性。心灵术士最初须选择一个基本能系，从而获得附加的心灵能力，尽管他在相关属性值足够高时可以学习任何能系的异能。
　　　
　　{ Andor注：学习（发掘）异能并无属性要求，只在展现时须显能者的该异能所属能系关键属性值足够。 }
　
　　心灵术士可以在一枚水晶或宝石内注入自己的人格片断来创造一个灵晶仆(Psicrystal，类似魔宠)。灵晶仆拥有对心灵术士很有帮助的特殊能力。
　　　
　　心灵术士自身知识增长时便会获得新的异能。但是为了真正地认识自己，心灵术士不只需要内心的修行，同时也需要体格上的，所以他们通常会学习一点基本的战斗技巧并擅长简易武器。
　　　
　　阵营：对于心灵术士来说，灵能只是个人技艺。心灵术士们看起来都比较内向，所以他们稍稍倾向于中立阵营而非混乱或守序，但并非必须中立。均衡来看，心灵术士中既有善良的也有邪恶的。
　　　
　　宗教：由于特别关注他们自身的潜能，心灵术士们并未显著地信仰任何神祗。崇敬神祗的心灵术士们并无规律可循。(Psions who do revere a deity do not conform to any general choice.)
　　　
　　背景：注定要成为心灵术士的人们在幼年时就表现出精神力量的征兆，到了青春期这种表现变得特别显著，像出现莫名其妙的声音和光芒，陶器自己在空中乱飞，小物件无中生有地出现等都是很常见的。大部分拥有这种天赋的人都没有发掘自己的潜能，而另一些欣然接受了这种特殊能力。从那之後，崭露头角的心灵术士开始练习并改善他们的能力。如同舞弄法术的术士一样，因为朋友的畏惧和家人的误解，心灵术士们通常都很孤僻。有时，心灵术士也能找到其他有类似能力的伙伴并组成非正式团体甚至是聚集拥有心灵能力者的小型社区。
　　　
　　种族：灵能这种天赋不可预测，在任何普通种族中都有可能出现心灵术士。
　　
　
　　灵吸怪(illithids)非常可能成为心灵术士，因为他们的能力实际就是深层灵能本质的成分，他们甚至不须心灵术士等级就已拥有可怕的心灵力量。
　　
　
　　与其他职业的关系：心灵术士们欣赏与他们同样投身于自我控制的武僧团体。有些心灵术士看不起法师、术士这些施法者，因为他们觉得魔法只是心灵的拐杖，精神不完善的人才需要魔法。心灵术士常常和心灵武士互相戏弄，但他们的对抗很少导致彻底的冲突。
　　　
　　心灵术士的能力使他对任何冒险团队来说都是一个强有力的伙伴。不同能系的心灵术士在队伍中扮演着不同的角色。例如，心灵预言(Clairsentience)专家通过高难度侦察引领队伍，而选择心灵传送系 (Psychoportation)作为基本能系的心灵术士则能大大提高队伍的机动力。

游戏规则信息
GAME RULE INFORMATION
　
　　心灵术士拥有以下游戏数据。
　　
　
　　属性：每一项属性对心灵术士来说都可能至关重要，因为每一能系都有影响该能系异能使用的关键属性(见关于能系的边栏)。每一异能都属于六大能系之一，每一能系都有一项不同的关键属性。例如，敏捷是心灵传送系的关键属性。特定能系内的所有异能受同一项属性影响。
　　　
　　当心灵术士使用心灵异能时，称为“展现”，“展现异能”简称“显能”。欲展现任何异能，心灵术士的关键属性值必须达到10加该异能能级（就是说，某能系的关键属性值小于10的心灵术士无法展现该能系的异能）。心灵术士还必须拥有足够的灵能点支付异能费用。
　　　
　　选择属性值很重要的另一个原因是防御者对抗心灵术士攻击者的异能的意志豁免DC是由攻击者每次显能时决定的：1d20 + 该异能能级 + 关键属性调整值(详见第四章“灵能综述”)。
　　　
　　心灵术士还根据他的基本能系的关键属性值获得的额外灵能点（这点从不改变）。表1-2“心灵术士的额外灵能点”显示了基本能系拥有高属性值的心灵术士升级时获得的额外灵能点数。这些点数是加成的；因此，关键属性值18的5级心灵术士有 1 + 3 + 5 = 9 点额外灵能点。而关键属性值为15的8级心灵术士只有 1 + 3 = 4 点额外灵能点，和他较低等级时一样。额外灵能点只由24小时无波动的高属性值产生。
　　　
　　除了高属性值，心灵术士必须有足够等级才能获得该等级的额外灵能点。例如，心灵术士亚历珊卓（她的基本能系为心灵传送系）在3级时敏捷为16，足以在1级获得1点额外灵能点并在3级获得3点额外灵能点，总共4点。她达到5级时才能再获得5点，也就是总共9点。
　　　
　　如果你的人物关键属性值为9或更低，则无法展现出该属性相关的（也就是对应能系的）心灵异能。例如，若亚历珊卓的敏捷因中毒或心灵战斗而降为9，那么在被治愈前她都无法再展现任何即使最简单的心灵传送系异能。
　　　
　　阵营：无限制。
　　　
　　生命骰：d4。

能系
DISCIPLINES
　
　　心灵异能分为六大能系(Disciplines)，每系都有共同主题，如心灵创造系和心灵附魔系。每一道已知异能都属于六大能系之一。在创建人物的时候，心灵术士必须选择一个能系作为基本能系，基本能系一旦选定就不能改变。人物可以依据表1-1选择任何能系的心灵异能并且每异能能级还将知道基本能系的 1个额外异能。基本能系还决定了由哪项关键属性提供心灵术士额外灵能点(见表1-2)。

　　心灵自塑系(Psychometabolism)：改变心灵术士身体机能的心灵异能。选择心灵自塑系作为基本能系的心灵术士被称为百变者(Egoist)。力量是该能系的关键属性，并决定百变者的额外灵能点数。
　　　
　　心灵传送系(Psychoportation)：能使人物或物品时空旅行的心灵异能。选择心灵传送系作为基本能系的心灵术士被称为灵旅者(Nomad)。敏捷是该能系的关键属性，并决定灵旅者的额外灵能点数。
　　　
　　心灵转化系(Psychokinesis)：操控或改变物质能量的心灵异能。选择心灵转化系作为基本能系的心灵术士被称为操能者(Savant)。体质是该能系的关键属性，并决定操能者的额外灵能点数。
　　　
　　心灵创造系(Metacreativity)：能通过星界摄取物质，创造实体或半实体物品的心灵异能。选择心灵创造系作为基本能系的心灵术士被称为创形者(Shaper)。智力是该能系的关键属性，并决定创形者的额外灵能点数。
　　　
　　心灵预言系(Clairsentience)：揭示隐藏信息的心灵异能。选择心灵预言系作为基本能系的心灵术士被称为洞察者(Seer)。感知是该能系的关键属性，并决定洞察者的额外灵能点数。
　　　
　　心灵附魔系(Telepathy)：能够同其他有意识的生物建立精神联系或进行心灵控制的心灵异能。选择心灵附魔系作为基本能系的心灵术士被称为传心者(Telepath)。魅力是该能系的关键属性，并决定传心者的额外灵能点数。

职业特性
Class Features
　
　　以下均为心灵术士的职业特性。
　
　
　　擅长武器与防具：心灵术士擅长所有简易武器，他们不擅长任何盔甲与盾牌。防具不会干扰心灵异能的使用，但当穿戴不擅长的盾牌或盔甲时，攻击检定和以下技能会遭受防具检定减值：“平衡”、“攀爬”、“脱逃”、“躲藏”、“跳跃”、“潜行”、“扒窃”和“翻滚”(关于防具检定减值见《玩家手册》第七章)。{ Andor注：穿戴擅长防具时所列技能也同样有减值。 }
　　　
　　异能：心灵术士展现异能。心灵术士能选择的异能非常有限，然而享有极大的灵活性。1级的心灵术士只知道所选的2个零级异能（或称为“天赋异能”）和0个一级异能再加上基本能系的每能级1个额外异能，一共3个零级异能和1个一级异能。心灵术士可随着升级发掘出一个或多个潜藏的异能，如同表1-1“心灵术士”所标明的，心灵术士可从任何能系选择异能，但表1-1每列中的“d”表示基本能系的额外异能。“--”表示并无已知异能，“d”或数值表示心灵术士在该级可以选择若干异能。
　　　
　　已发掘的异能可以是心灵术士异能列表中的一般异能 (见第五章“异能详述”)或心灵术士通过研习领悟到的特殊异能。例如，一个心灵术士有一枚注有（心灵术士异能列表上所没有的）特殊异能的灵能石，他便可以在升级获得新异能时选择这道异能，当然他必须达到展现这道异能的等级。请注意，在任何情况下心灵术士都不可以用这种方法习得超过“表1-1”所规定数量的心灵异能。
　　　
　　欲展现某一能系的异能，心灵术士的关键属性值至少要达到10 + 异能能级。（心灵攻击和防御模式不受这个限制，也不具有能级。）抵抗心灵术士异能的豁免检定DC为1d20 + 异能能级 + 心灵术士的关键属性调整值，该1d20由心灵术士攻击者在展现异能时掷骰。(心灵攻击和防御模式使用特定修正值，详见第四章。)
　　　
　　另外，在展现某些异能时可以支付更多灵能点以增强效果。第三章“专长”这部分详述了很多增强心灵异能的专长。
　　　
　　注意：一名心灵术士在升级时可以选择发掘较低能级的异能来代替正常能级的异能。因此，刚从8级升到9级的心灵术士可以选择习得一个三级或更低能级的异能来代替四级的。
　　　
　　灵能点：心灵术士每日可展现的异能数量由他的可用灵能点数决定，但他不用预先准备异能——他只要在展现异能时支付相应的灵能点费用。表1-2中显示的额外灵能点可增加心灵术士的每日可用灵能点数。例如，传心者艾达波迪在1级时可得到2点每日灵能点数，加上足够的魅力（他的基本能系的关键属性）带来的1点额外灵能点，他还选择了“内心强健”专长，所以还可得到1点灵能点，也就是一共4点每日灵能点数。
　　　
　　零级异能：又称“天赋异能(talents)”。对于天赋异能，有特殊的灵能点花费规则。心灵术士每日可免费任意展现 职业等级 + 3 道天赋异能，用尽了这个次数以後，该日内每展现一道零级异能需花费1点灵能点。因此，一个1级心灵术士每日可免费展现4道天赋异能，此後若在同一天内还要展现零级异能则每道需花费1点灵能点。
　　　
　　心灵战斗模式：1 级时，心灵术士可以选择习得10项心灵战斗模式中的5项。(心灵战斗模式的具体描述和设定见第四章“灵能综述”。)此後，在3级、5级、7级、9级和11 级时又可分别再获得1个心灵战斗模式(如表1-1所示)。五项攻击模式为“自性鞭击”、“本我暗示”、“心灵爆破”、“心灵穿刺”和“心力榨取”。五项防御模式为“心灵空白”、“智力堡垒”、“精神障壁”、“思维之盾”和“坚固意志”。
　　　
　　心灵攻击和防御模式在对付灵能人物和生物时特别有效，但也可影响非灵能生物。在某些方面心灵战斗模式很像异能，但它们具有异能所没有的特殊性质。
　　　
　　心灵战斗模式有不同的展现费用，这里提醒低级心灵术士选择低费用的战斗模式。例如，1级心灵术士可以选择“心灵爆破”，但高达9的灵能点费用使其在近几级内都难以使用。
　　　
　　灵晶仆：心灵术士能够创造一枚灵晶仆。创造灵晶仆的过程需要花掉一天，还需价值100金币的材料（通常作为灵晶仆基体的水晶或宝石至少价值总花费的50%）。灵晶仆形如一枚直径不到1吋的透明水晶石，其载有心灵术士自己的人格片断，因而与心灵术士有精神联系，可以增强心灵术士的部分能力。一名心灵术士同时最多只能拥有一枚灵晶仆。
　　　
　　心灵术士在创造灵晶仆时选择灵晶仆的种类，就是说，决定将自身哪一部分人格片断注入水晶石。当心灵术士升级时，灵晶仆的能力也随之提升，见表1-5“灵晶仆的特殊能力”。
　　　
　　灵晶仆被摧毁时，心灵术士必须进行强韧检定(DC=15)，若未通过则失去每等级200点经验值，通过则减半。但是心灵术士的经验值不会因此而减到0以下。例如，当艾达波迪的灵晶仆被索尔石怪吃掉时，他是有着3230点经验值的3级心灵术士。艾达波迪通过了豁免，因此将失去300点经验值，使他的经验值低于3000而掉回2级(《地下城主指南》中有关于失去等级的规则)。一旦灵晶仆被摧毁，心灵术士必须等到六个月後才能再次创造新的灵晶仆。
　　　
　　起始金钱：3d4×10金币（除非使用起始套组）。
　　Starting Gold: 3d4x10 gp (unless using a starting package).
