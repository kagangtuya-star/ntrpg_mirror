Great Dismal Library

你说不清这到底是一座图书馆还是一座寺庙。高大紧闭的窗户，昏暗摇烁的烛光，穿着长袍戴着罩帽的管理员让你想起印记城里的Dustman，

阿斯特纽斯在帕兰萨斯图书馆里记载过去和现在，智者阿兰多在烛堡里撰写未来，苦力们在大阴暗图书馆里挖坑......呃，是的，这些僧侣似的家伙都被称作“苦力众”，他们喜欢在图书馆里四处挖坑，有些良心没有泯灭的家伙过些时候会主动把坑填上，但大多数坑就这么放在那，有时两个苦力会挖到同一个坑里，他们管这个叫“撞车”......

好罢不管那些乱七八糟的事，这地方就实在太暗了！你决定先向管理员们提出这个意见，很快你得到了答复：请自行调节显示器的亮度和对比度，建议购买非礼扑LightFrame系列产品.......

灵晶仆(PSYCRYSTALS)
 { Andor注：以下由 Andor 翻译。 }
　
灵晶仆(PSYCRYSTALS)

　
　　除了主人，在任何人看来灵晶仆都是值1金币的普通宝石。然而根据植入石头内的不同人格片断，灵晶仆能给予主人某种特殊的能力（见表1-4“灵晶仆”）。物理上，灵晶仆AC为13，硬度8，生命值20。灵晶仆若有损伤，则以每日2d4点生命值的速率“重生”晶体结构。生命值为0的灵晶仆被摧毁。
　　　
表1-4：灵晶仆
─────────────────────────────────
灵晶仆人格　特殊能力
─────────────────────────────────
巧匠　　　　主人的手艺检定获得+2加值。
恶霸　　　　主人的威吓检定获得+2加值。
懦夫　　　　主人的躲藏检定获得+2加值。
友善　　　　主人的交涉检定获得+2加值。
英雄　　　　主人的强韧豁免获得+2加值。
惯谎　　　　主人的唬骗检定获得+2加值。
细心　　　　主人的搜索检定获得+2加值。
顾盼　　　　主人的侦察检定获得+2加值。
沉着　　　　主人的平衡检定获得+2加值。
坚毅　　　　主人的意志豁免获得+2加值。
智者　　　　主人的任一项已会的知识检定获得+2加值；选定後不可更改。
专心　　　　主人的专注检定获得+2加值。
静谧　　　　主人的潜行检定获得+2加值。
同感　　　　主人的察言观色检定获得+2加值。
─────────────────────────────────
　

　
　　灵晶仆人格描述：所有灵晶仆都有独特的人格。心灵术士在1级时只能强烈感觉到灵晶仆的人格，但在高级时灵晶仆可以自由地通过心灵感应向心灵术士提供建议，这些建议强烈倾向于灵晶仆自己的世界观。主人总是能从他的灵晶仆中看见一部分自己，即使是由于夸大而扭曲的自己。
　　　
　　巧匠(Artiste)：这小片人格毫不谦虚地表达自己在几乎所有职业上有惊人的天赋，例如舞蹈、歌剧和烹饪。同样多的誓言和自负充满了该宣言。
　　　
　　恶霸(Bully)：有时坏品格对某些事而言是倒是优点。恶霸人格很讨厌，它蛮横而夸夸其谈并且非常固执地认为自己是对的，但心灵术士可以不时善用这种影响。
　　　
　　懦夫(Coward)：健全的自卫本能深藏在每个人的心底。当懦弱的要素被捕捉并放大时，可以帮助心灵术士在危险情形下寻找安全港。然而，这种灵晶仆过分热衷于催促心灵术士逃跑并躲藏，即使只是在一般危险的情形下。
　　　
　　友善(Friendly)：这小片人格乐于助人，甚至是敌人。心灵术士主人得忍受那针对他的硬心肠的说教，结果就是心灵术士会更乐于以非暴力方式解决对手。
　　　
　　英雄(Hero)：我们都想成为英雄，灌输了这部分人格的灵晶仆便是这种感觉的人性化。它惯于激励心灵术士进入危险境遇，并回报给主人以在任何情形下的强韧身体。
　　　
　　惯谎(Liar)：爱说谎的灵晶仆会情不自禁地说谎，即使是对深谙它何时在说谎而何时在说实话的主人。当心灵术士打算说谎的时候它的说谎方式就能帮上忙了。
　　　
　　细心(Meticulous)：细心的灵晶仆花费大量时间辨别重要细节和无关紧要的细节。它对自己的描述是“对礼节一丝不苟，而且极端注重细节”。这种特点虽然有时几乎令人发狂，但对于试图仔细研究所处环境的心灵术士来说是很有用的。
　　　
　　顾盼(Observant)：“你注意那个了么？”是灵晶仆最常告诉主人的。心灵术士常常早已看见灵晶仆提示的东西并由于不重要而忽略了。然而有时这种特点会提醒心灵术士注意到忽略了的东西。
　　　
　　沉着(Poised)：时时注意自己的身体和环境的配合常常会导致失去更重要的想法。Not so for this psicrystal, which is perfectly poised and sure of itself, if not its owner. 它总是宣扬平衡感的重要性并随时准备行动，它的影响倒确实能在平衡比较重要时给予心灵术士帮助。
　　　
　　坚毅(Resolve)：该灵晶仆正拥有这种所有人都非常想要的品质。它的意志和信仰非常坚固，坚定和决心体现在它的建议和意见中。或者，用心灵术士的话说，他是这样顽固。然而，这种坚定不移的信念帮助心灵术士进行一切意志对抗。
　　　
　　智者(Sage)：这小片人格试图成为主人的良师，即使它本身就是从主人的精神中抽出的。然而，脱离了其他人格类型，智者认为自己是精神和哲学上最重要的大师，而不提及自己专精的知识领域。智者灵晶仆超出专业以外的建议常常是不可靠的。
　　　
　　专心(Singleminded)：植入该灵晶仆的是忽略环境和情感因素而专注手头工作的能力。有时心灵术士很难引起专注在某件特定任务上的灵晶仆的注意，但它的影响也扩大了心灵术士自己的专注能力。
　　　
　　静谧(Sneak)：可以用比呼吸声还轻的脚步声出现时为何要用沉重的脚步声引人注意呢？全身心信仰沉默是金的作用的静谧灵晶仆大概会这么问。它惯于使主人和主人的同伴高度安静。然而，它在安静移动方面的专业性不可否认。
　　　
　　同感(Sympathetic)：这种性质使灵晶仆设身处地地看穿其他生物的心思。不幸的是，灵晶仆花大量时间分析心灵术士及其同伴，评论他们的心理状态，并且对他们行动的适当性发表判断，这种能力在心灵术士试图确定潜在敌人的动机时非常有用。
　　　
　
　　心灵术士等级：心灵术士主人的等级。
　　　
　　灵晶仆能力描述：所有的灵晶仆都有基于主人等级的特殊能力。这些能力是累计的。
　　　
表1-5：灵晶仆的特殊能力
────────────────────
心灵术士等级　灵晶仆智力　特殊能力
────────────────────
１─２　　　　６　　　　　超视、情感连结
３─４　　　　７　　　　　心灵连结
５─６　　　　８　　　　　自行移动
７─８　　　　９　　　　　与其他生物交谈
９─１０　　　１０
１１─１２　　１１　　　　异能抗力
１３─１４　　１２　　　　视觉连结
１５─１６　　１３　　　　引导异能
１７─１８　　１４
１９─２０　　１５
────────────────────
　
　
　　智力：灵晶仆的智力值。灵晶仆是聪明的，但他们的人格特性会影响所有情感、心灵和语言交流。
　　　
　　超视(sighted)：虽然没有物理感觉器官，但灵晶仆可以通过心灵感应如同拥有视觉和听觉的生物那样感知自身周围环境。黑暗（即便超自然黑暗）不影响超视，超自然静默也不影响。然而灵晶仆仍不能看出隐形或灵体生物。灵晶仆超视距离为40呎。
　　　
　　情感连结(empathic link)：心灵术士与一哩内的灵晶仆有情感连结。心灵术士无法通过灵晶仆的超视能力观察，但双方可以互通情感。灵晶仆和主人间的情感连结仅限于互通强烈的感觉，例如自信、害怕、怀疑、决心等等，因此总是有可能误解。
　　　
　　心灵连结(telepathic link)：心灵术士与一哩内的灵晶仆有心灵连结。心灵术士无法通过灵晶仆的超视能力观察，但双方可以心灵沟通。心灵沟通允许真正的交流。例如，放在远处房间里的灵晶仆可以将房间里发生的事情转告主人。灵晶仆懂得它们的心灵术士主人会的所有语言。
　　　
　　自行移动(self-propulsion)：心灵术士支付1点灵能点便可使他的灵晶仆长出星质蜘蛛腿并维持一天。这些腿给予灵晶仆30呎速度。灵晶仆可以在墙上和天花板上以20呎速度像蜘蛛一样爬行。到期後或灵晶仆受到穿透硬度後的至少1点伤害时这些腿就消失不见了。
　　　
　　与其他生物交谈(speak with other creatures)：灵晶仆可以使用主人所会的任一种语言和60呎内的主人、其他生物或其他灵晶仆精神沟通（其他生物必须大声说话来回应，而主人和其他灵晶仆则不用）。对熟悉心灵术士的人来说，灵晶仆的精神嗓音听起来很像其主人，然而，所有的交流都受到灵晶仆独特人格的影响。
　　　
　　异能抗力(power resistance)：灵晶仆的异能抗力(PR)等于主人等级加5。敌人如想展现异能以影响灵晶仆，则必须进行施法者等级检定(1d20 + 敌人的显能者等级)，结果至少要等于灵晶仆的异能抗力方可。除非采用变体规则“灵能异于魔法”(见第四章“灵能综述”)，灵晶仆的法术抗力(SR)等于其异能抗力。
　　　
　　视觉连结(sight link)：每日一次，只要灵晶仆没有离开主人超过一哩，在一小时主人都可以看见灵晶仆所见到的。
　　　
　　引导异能(channel power)：心灵术士可以经由一哩内的灵晶仆展现异能。灵晶仆被视为该异能的发出者，所有的距离都按其所在位置计算。经由灵晶仆引导异能时，心灵术士须支付费用并展现该异能。他会遭受可能的借机攻击或展现异能的其他相关风险。
