Great Dismal Library

你说不清这到底是一座图书馆还是一座寺庙。高大紧闭的窗户，昏暗摇烁的烛光，穿着长袍戴着罩帽的管理员让你想起印记城里的Dustman，

阿斯特纽斯在帕兰萨斯图书馆里记载过去和现在，智者阿兰多在烛堡里撰写未来，苦力们在大阴暗图书馆里挖坑......呃，是的，这些僧侣似的家伙都被称作“苦力众”，他们喜欢在图书馆里四处挖坑，有些良心没有泯灭的家伙过些时候会主动把坑填上，但大多数坑就这么放在那，有时两个苦力会挖到同一个坑里，他们管这个叫“撞车”......

好罢不管那些乱七八糟的事，这地方就实在太暗了！你决定先向管理员们提出这个意见，很快你得到了答复：请自行调节显示器的亮度和对比度，建议购买非礼扑LightFrame系列产品.......

[心眼]新的灵能专长、异能和怪物
日期：2004-08-25 　作者：Windslice 　来自：DNDFANS.org 　点击: 4954
 
 

 

异能详述：A字头

心眼：异能详述（字母序）

 

快速代谢（Accelerated Metabolism）

  心灵自塑系（力量）

  等级：心灵术士 3

  异象：听觉（这里原文是MU，而根本就没有MU这个异象，所以估计应该是AU）

  展现时间：一整轮

  距离：个人

  目标：自己

  持续时间：1分钟

  灵能点：5

 

  在持续时间内，你每轮回复1点HP。

 

强酸激射（Acid Splash）

  心灵创造系（智力）[酸]

  等级：心灵术士 0

  异象：视觉

  展现时间：单动作

  距离：近距（25英尺+5英尺/每2等级）

  目标：单个生物或物体

  持续时间：立即

  豁免检定：无

  异能抗力：可

  灵能点：1

 

  （同3R同名法术）你朝目标发射一小团酸液，如果成功以远程接触攻击击中对手可以造成1D3酸伤害。

 

改变气味/味道（Alter Smell/Taste）

  心灵附魔系（魅力）[影响心灵]

  等级：心灵术士 0

  异象：精神

  展现时间：单动作

  距离：近距（25英尺+5英尺/每2等级）

  目标：单个生物或物体

  持续时间：立即

  豁免检定：意志，通过则无效

  异能抗力：可

  灵能点：1

 

  你 可以改变一个目标的气味或味道，受该异能影响的生物可能会稍微更讨人喜欢或更让人厌恶一点。不过这种变化总是很微小的。例如，你可以把一个新鲜橘子的气味变成已经腐烂的气味，或是把一瓶好酒变成醋的味道。如果目标通过了意志豁免检定，那么在这一天内你都无法再尝试改变他的气味或味道。

 

星界蜘蛛（Astral Spider）

  心灵创造系（智力）

  等级：心灵术士 0

  异象：物质，视觉

  展现时间：单动作

  距离：近距（25英尺+5英尺/每2等级）

  持续时间：1分钟/等级（解消）

  灵能点：1

 

  一个2尺宽有着微弱银光的球型星质体出现在异能距离内。它有六只蜘蛛般的腿和一个6尺长像鞭子一样发着微弱绿光的尾巴。

  星界蜘蛛能够以20尺的速度移动或者像一般蜘蛛一样以10尺的速度在墙壁，天花板攀爬。那条尾巴可以抓住并移动如钥匙，宝石等至多1磅的物体。星界蜘蛛的AC为13（-5敏捷，+8体型），具有3点HP。

  只要持续专注并且一直存在效果线，星界蜘蛛就会完全按照你的意志行动。星界蜘蛛不能进行攻击或与其他生物交流。持续时间结束时星界蜘蛛便会无害的消失。

 

一级星界仆役（Astral TaskerⅠ）

心灵创造系（智力）

  等级：心灵术士1

  异象：视觉，物质

  展现时间：1整轮

  距离：近距（25英尺+5英尺/每2等级）

  效果：造出一个星界仆役

  持续时间：1小时/等级

  豁免检定：无

  异能抗力：不可

  灵能点：1

 

  星界仆役是由星质构成的活化物体，可以用来完成一些简单的任务。它们通常看起来比较矮小，没有什么特征，类人造型，就像星界构装体。一个成功的手艺（雕刻）检定[DC=10+（2*星界仆役等级）]可以重塑星界仆役的外型（具体见下文关于塑型的描述）。

  星 界仆役可以取东西，打开普通的门，柜子，抽屉，跟随特定队员，搬箱子，提口袋之类的。一个星界仆役一次只能执行一项任务，并且在你让它停止之前它会一直不断的执行这个任务。例如，一个星界仆役正扛着一个桶跟在心灵术士后面，如果前面有扇关着的门，星界仆役就会停在门前不动，你必须先告诉它把桶放下，然后再 开门。星界仆役可以触发压力要求只少100磅的压力板。

  任何情况下，星界仆役都不可进行攻击，但它们可以正常的进行豁免检定，它们不会死亡，只是在生命值降到0时被人道毁灭。只有在异能范围内时，你才可以对星界仆役下命令，不过在没有收到新命令前，它会一直尽全力执行之前的最后一个命令直到持续时间结束或是被毁灭。下面的怪物部分有星界仆役的详细数据。

  这个异能造出一级星界仆役。

 

二级星界仆役（Astral TaskerⅡ）

心灵创造系（智力）

  等级：心灵术士2

  异象：视觉，物质

  展现时间：1整轮

  距离：近距（25英尺+5英尺/每2等级）

  效果：造出一个或更多的星界仆役，每两个之间的距离不超过30英尺

  持续时间：1小时/等级

  豁免检定：无

  异能抗力：不可

  灵能点：3

 

  如同一级星界仆役，，只是你可以创造1个二级星界仆役或者是1D3个1级星界仆役。
