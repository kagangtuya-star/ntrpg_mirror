法兰溯历（FT）

 -118 FT 苏人开始了征服与繁盛之年代。东南方的法兰族（地狱熔炉山脉（Hellfurnaces）以西），东方的奥尔迪安族，南方的克西人（最初来自阿那克瑞的克西人的后裔）和北方的巴克伦人，以及西方许多没有名字的小部落都沦陷在苏的铁蹄之下。奥瑞克大陆的半个西边都完全在苏的控制之下。（3247 SD / -2269 CY）

-115 FT 法兰族在贝欧瑞（Beory）、培罗（Pelor）和拉奥（Rao）的保护下大量逃离自己的家园，开始了翻越地狱熔炉山脉的危险旅程。他们北上进入之后被称作法兰尼斯（Flanaess）的奥瑞克东部，成为这片区域的首批人类住民。他们很快就受到了原住民的接受。（3250 SD / -2266 CY）

1 FT 第一个法兰人的城市“罕若达哈”（Haradahagh）在罗特米尔山脉（the Lortmils）建立。这是法兰溯历元年。（3365 SD / 2312 OC / -2151 CY）

161 FT 法兰人法师加拉普-德瑞德（Galap-Dreidel）建造了因弗内斯（Inverness）来保护他的“灵魂之石”（Soul Gem）。（3526 SD / -1990 CY）

219 FT 法兰族第一次有了关于南方信仰奇怪神祉的奇怪城市的报告。这些人（根据法兰人的资料记载）自称艾尔美克人（Almek）（在通用语里即奥尔美克，Olmec）。（3584 SD / -1932 CY）

404 FT 位于埃莱多西面的一个法兰人小部落，乌尔-法兰族（Ur-Flannae），给灰精灵王国送去作为人质的奴隶以此换取土地权。第七任高王，希尔索尼奥•闪盔（Gilthonial Glitterhelm），特别惊讶于一个有前途的年轻人敏锐的头脑，他的名字叫维克纳（Vecna）。维克纳在精灵光辉伟大成就的激励下开始学习魔法。他在图书馆里发现了一本来自苏的名为《提罗若普之命运》（The Fate of Tilorop）的书，他开始偷偷信仰萨瑞兹顿（Tharizdun）。（3769 SD / 2716 OC / -1747 CY）

435 FT 维克纳向希尔索尼奥请求回故乡探望。该请求被批准。（3800 SD / 2747 OC / -1716 CY）

440 FT 维克纳在奈尔戴夫（Nyr Dyv）中央建起一座黑塔。他要求成为乌尔-法兰部落的酋长，并在决斗中用魔法杀死了前任酋长。（3805 SD / 2752 OC / -1711 CY）

411~468 FT 维克纳用魔法对精灵隐藏了自己的踪迹。他研究了《提罗若普之命运》并受到想同精灵一样长寿的贪婪所驱使。他模仿灰精灵建起高塔并且用乌尔-法兰的人民作实验，开始“究极死亡大转化”（Ultimate Solution to Death）。许多不死生物被制造出来。（3806~3833 SD / 2753~2780 OC / -1710~1683 CY）

480 FT 维克纳完成了转变成巫妖的所有条件，然后将自己变成了巫妖。他来到北方平原并带回了许多类人生物加入他的阵营。他把他的人民变得堕落而扭曲，通过这些培养，如今他有了一支人数远远超过东方精灵的大军。（3845 SD / 2792 OC / -1671 CY）

496 FT 维克纳锻造了那把后来传给凯斯（Kas）的剑。（3861 SD / 3328 OC / -1655 CY）

482~506 FT 维克纳向希尔索尼奥进军。他把惨败的军队变成不死生物秘密军。他的类人生物、不死生物和法兰族联军渐渐将灰精灵赶回了他们的山中要塞。（3848~3883 SD / 2794~2820 OC / -1669~1643 CY）

507~576 FT 希尔索尼奥军被一次来自狮鹫山脉西侧平原的闪电攻击所驱赶。维克纳的军队一直扫荡到奈尔戴夫的东岸及北方地区。他们占领的平原直抵狮鹫山脉脚下。（3874~3943 SD / 2821~2900 OC / -1644~1575 CY）

585 FT 塞莱涅派遣了一支军队援助希尔索尼奥。虽然最初的合作取得了一定成功，但维克纳呼唤了萨瑞兹顿的力量并释放出一股高热魔法力——辉煌沙漠就是这么诞生的。精灵方面伤亡惨重，精灵军陷入恐慌之中。他们撤退回塞莱涅，但维克纳没有追击。他的目光集中在东方。（3763 SD / 2906 OC / -1557 CY）

604 FT 维克纳的军队占领了第一座灰精灵在山区中的城市。这是400年战争的开始。（3982 SD / 2925 OC / -1538 CY）

921 FT 希尔索尼奥军慢慢地在维克纳前撤退，没有援助。狮鹫山脉中仅剩5座灰精灵城市还未被维克纳摧毁。（4299 SD / 3242 OC / -1221 CY）

988~994 FT 希尔索尼奥离开王宫和他的军队一同向维克纳发动进攻并将他赶出了山脉。但一进入平原，维克纳就使出了他的新武器，一把用来自星辰的物质所煅造的黑剑。维克纳和希尔索尼奥展开了一对一的战斗并杀死了他。维克纳的军队歼灭了精灵军并再度挺进山脉。他们占领并毁坏了精灵的城市，并按维克纳的理念重塑了它。但是有四个城市却始终没被发现。首都伊尔瑞艾登，灰精灵建筑艺术的顶峰和权力中心化作一片废墟。埃莱多精灵王国、高王的宝座也就此从世界上消失。（4362~4368 SD / 3309~3315 OC / -1154~1148 CY）

995 FT 凯斯成为维克纳的首席副官。维克纳将自己的传说之剑赐予他。（4369 SD / 3310 OC / -1156 CY）

997 FT 夏星城被发现后，达那卡瑞王子（Price Darnakurian）使用他的暗黑之刃奋起抵抗维克纳的入侵。维克纳的军队被摧毁。夏尔菲儿女王（Queen Sharafere）与她幸存的子民们逃往南方岛屿。（4371 SD / 3318 OC / -1145 CY）

1000 FT 由于在进攻夏星城时耗费了大量法力，维克纳变得虚弱起来。一直窥视其地位的凯斯找到机会除掉他。维克纳的身体被摧毁，只留下一手一眼。凯斯也死于这场战斗。维克哪的一些追随者们创立了“维克纳教派”。维克纳帝国崩溃。（4374 SD / 3321 OC / -1151 CY）

这之后出现了许多法兰小王国，但都无法同维克纳统治下的乌尔-法兰曾经拥有的辉煌相提并论。许多北方的法兰人既害怕精灵军队，又害怕再出现另外一个巫妖王的残暴统治，便逃出了城市回到部落统治的生活方式中。在这场大规模的瓦解中，只有一个名为坦恩（）的能干塞恩酋长（Theign）统治下的土地除外。他尽力不让自己的人民溃散，虽说这个地区经常会有游荡怪兽和不死生物的入侵。

1117 FT 萨姆法兰王国（Flanae Kingdom of Sulm）在辉煌沙漠中建立。（4482 SD / -1034 CY）

1583 FT 沙塔杜斯（Shattados），萨姆的法师国王和他的所有人民都被变成了蝎人（Scorpion Men）。（4948 SD / -558 CY）

1693 FT 遵从先知的警告，许多奥尔迪安人开始向东迁徙。他们接触到了法兰人并与之发生了冲突。（5058 SD / -458 CY）

1704 FT 皇帝之子、贝塔的高阶司祭、巫师泽里发•安德-佐尔（Zellifar ad-Zol）与其父决裂，带领效忠他的8000多苏人向东逃出帝国。这场逃亡来势凶猛，再加上精良的魔法让沿路的奥尔迪安人四散奔逃，没有迁徙的奥尔迪安人不得不移居，转而围攻法兰人。佐尔氏人继续东进。（5069 SD / -447 CY）

1706~1728 FT 佐尔氏人驱散了挡在路上的法兰人，向南进入提凡诺特半岛（Tilvanot Peninsula）。三个追击的家族没能发现魔法通道，北上遭遇了重新聚集起来的奥尔迪安人和害怕的法兰人。他们沉重打击了这些苏人贵族并将他们赶回了南方。（5071-5093 SD / -493 ~ -445 CY）

文化特性：奥瑞克大陆的法兰人十分类似地球欧洲的日耳曼人和凯尔特人。在向东迁徙之前，他们一直过着部落式的生活。即使后来在达到其文化顶峰之时，他们身上依然保留着许多部落式的特征。刚逃至现在被称为法兰尼斯的东奥瑞克大陆时，这里的类人种族非常友善的对待他们。但自从建立了他们的第一批城市，法兰人便开始和那些类人种族疏远了。
精灵王国埃莱多（Aliador）允许法兰人定居，得到的回报是敌意和战争。在维克纳的统治下，法兰人变成了一个一元化的民族。然而对乌尔-法兰人（Ur-Flan）（生活在东北部）来说，维克纳的统治永远地改变了他们。除了居住在坦恩的人以外，乌尔-法兰人都回复到原始的流浪生活，好像地球上美洲的土著。西部的法兰人被入侵文化吸收了，就像欧洲的罗马人和盎格鲁-萨克逊人先后两次吸收凯尔特人一样。

（法兰历史作者Steven B. Wilson，并参考了Gary Gygax的成就，World of Greyhawk(TSR)以及Vecna Lives(WGA4)，Rary the Traitor(WGR3)，The Dungeon Master’s Guild, 1ed.(TSR)，Greyhawk Adventures(TSR)。）
