巴克鲁尼什纪元（BH）

 

巴克鲁尼什纪元（BH）

 

 -1096 BH 北方平原的游牧部落第一次与苏帝国接触。他们开始依靠同帝国做生意生活。用成群的动物、马、布料和掏器换取精良的武器以及“文明人”的装饰品。（1760 SD / -3756 CY）

-32 BH 伊斯图丝（Istus）的先知，圣人艾尔-巴克伦-巴尔-加什（El-Baklun-bar-Gash, the Holy Man）预见了乌德鲁坎卡湖（Udrukankar）畔的神圣石碑。他来到那个地方并得到了许多人的支持。（2834 SD / -2692 CY）

1 BH 北方游牧部落在圣人艾尔-巴克伦-巴尔-加什的领导下发现或是建造（不明是哪一种）了一组奇怪的环状列石，并命名为托瓦格•巴拉古（Tovag Baragu），在巴克鲁尼什语中的意思是“世界的肚脐”。巴克伦人完成该地的时间被定为巴克鲁尼什纪元的开始。（2856 SD / -2659 CY）北方的游牧民族认为这是他们最神圣的地区，并开始自称为巴克鲁尼什或者巴克伦人。住在离这儿最近的雅米尔家族（Yamir）放弃了流浪的生活，开始充当圣地的保护者。

391 BH 第一次有记录的“奴隶掠夺”（Slavery Raid）。苏人军队俘虏了一整个巴克伦家族，把他们带回苏帝国进行奴役。（3247 SD / -2269 CY）

596 BH “惊世背叛事件”（Great Betrayal）。在一次商人集会上同七个游牧部落交涉之后，苏人奥迪亚佛（Odiafer）攻击了商人们，试图夺取他们的货物并奴役他们。七个家族聚集起所有力量反抗至最后一人倒下也没有投降。（3452 SD / -2064 CY）

605 BH 苏人军队进军北部平原并宣告占领。任何反抗的部落都遭到了镇压。苏的第一个北方要塞建立（3461 SD / -2055 CY）在接下来的600年中巴克鲁尼什一直奴属于苏。

1115 BH 巴克伦人在一个游牧部落酋长，安利-本-奥那（Ali-ben-Onar）的带领下试图摆脱苏的奴役。经过一场战争后一件苏人用来囚禁巨灵的物品被夺了过来。巴克伦人威胁说要使用这件魔法物品才让战争得以结束。苏人在“他们的”北方版图上割让了一大块给巴克伦人。（3971 SD / -1545 CY）

1121 BH “第一次胜利”。与苏交涉之后，巴克伦人宣称北方的一大片土地为他们所有。艾米尔家族（Amir）与苏坦家族（Sultan）聚集起来并选举宣告安利-本-奥那为“巴克伦全族之哈里发”。（3977 SD / -1540 CY）

1144 BH “弗鲁兹提战争”（Fruztii）。巴克鲁尼什击败了苏人贵族弗鲁兹提家的军队，巩固了他们在北方的统治。（4000 SD / -1516 CY）

1150 BH 伊斯图丝坦（Istustan）作为巴克鲁尼什的中心城市开始建设。许多游牧部落也开始在他们的故土上发展城镇，作为定居点。安利死后，他的侄子，莫希尼德-本-萨菲尔（Mohinid-ben-Saphir）被宣布为“巴克伦之哈里发”。（4006 SD / -1510 CY）

1224 BH 南方巫师戒约，一群赞雅（Xan Yae）的信徒利用他们的神术之力开始发展出“精神训诫”。（4080 SD / -1428 CY）

1381 BH 阿尔阿科巴（Al’Akbar），伟大的司祭与先知，创造出了他的圣杯（cup）和圣星（star）。（4237 SD / -1279 CY）

1494 BH 汝欧肯（Zuoken）的第一所修道院建立。精神肉体训诫开始严格实施。（4350 SD / -1166 CY）

1852 BH 据称如果皈依一种精神教条就能帮助击败苏人，因此莱马拉（Ramala）的汝欧肯修道院的一位导师离开了修道院向东跋涉去寻找灰精灵的国度学习他们的魔法。他再也没有回来。（4705 SD / -808 CY）

1932 BH 疫病在巴克鲁尼什的土地上蔓延，这是“疫灾期”的第一年。（4788 SD / -728 CY）汝欧肯信徒的修道院式训练流传开来，因此疫病带来的损失没有南方那么严重。但是许多巴克伦人将其归咎于苏并开始宣称应该清洗南方地区。

2033 BH 在哈利-本-乌西夫（Hali-ben-Usif）的带领下，巴克鲁尼什开始了第一次有组织的侵略苏。（4889 SD / -627 CY）在之后的一百年中，巴克鲁尼什渐渐繁荣起来，而苏则因为贵族间的混乱与不和而渐渐衰落。巴克鲁尼什朝廷富裕又兴盛，北方奥尔迪安人的土地沦陷在巴克伦人的统治下。

2157 BH 佐莱克斯家族的王子，祖尼德•安德-佐尔（Zunid ad-Zol）加冕为苏人之帝后，巴克鲁尼什从帝国撤回了他们的大使。（5012 SD / -503 CY）

2175 BH “大战役”爆发。哈里发哈兰-本-拉米夫（Haran-ben-Ramif）向萨哈特隘口派出一支军队，在那儿九千巴克伦人遭到屠杀。伊斯图丝的圣先知德芙拉（Devrah）预言了苏人的土地将被清洗得“如同沙漠般纯净”。（5021 SD / -485 CY）

2194 BH 巴克伦人和苏人都开始向山脉东部挺进。他们首次在战斗中使用了类人生物佣兵团。（5050 SD / -466 CY）

2211 BH 安瑞什-本-阿里夫（Arish-ben-Alif），流浪的先知警告了巴克鲁尼什的灭亡。他被伊斯图丝的高阶女司祭阿芙拉（Aphra）宣布为有罪。哈里发纳默什-彬-詹米什（Namesh-bin-Jamish）下令逮捕安瑞什，但他逃走了。（5067 SD / -449 CY）

2212~2215 BH 安瑞什徘徊在巴克鲁尼什领土的西部边境并让六个家族相信了将来的灾难。他们向东迁徙至法兰尼斯的北部地区。（5068~5071 SD / -448~-445 CY）

2230 BH 安瑞什回到朝廷试图劝服塔纳-彬-纳默什（Talna-bin-Namesh）带领巴克伦人民向东迁徙。安瑞什被处以死刑，活活被淹死。（5086 SD / -430 CY）

2235 BH 巴克鲁尼什军队开始入侵苏帝国的北方行省。由于内乱和追击家族的离开，苏人实力被大大地削弱了。在巴克鲁尼什的猛攻下节节败退。（5091 SD / -425 CY）

2238 BH 苏的巫师们聚集起他们的法力施放出了“律令毁灭”（Invoked Devastation）。没有一个巴克鲁尼什城市在这股魔法力前幸免。但巴克鲁尼什的牧师与巫师集合在托瓦格•巴拉古，启动了囚灵石的奥术力量，汇集他们最神圣之地的力量，再加上他们自身的法力，用“净炎之雨”（Rain of Colorless Fire）予以反击。这次魔法力的大释放所残留下来的就是现在的尘埃之海（Sea of Dust）和干涸之原（Dry Steppes）。（5094 SD / -422 CY）巴克鲁尼什哈里发帝国毁灭。但是雅米尔家族及听从安瑞什的劝告东迁的6个家族得以幸存。

2254 BH 东方的六大巴克鲁尼什家族中有四个投票决定回到故乡。凯提家族（Ketti）只前进到了图斯曼山地（Tusman Hills）以东，图斯米家族（Tusmii）、泽发家族（Zeifa）和埃科贝瑞家族（Ekberri）则回到了古老的巴克鲁尼什土地上。留在法兰尼斯北部的两个家族与西方的家族渐渐失去了联系。干涸之原的不毛之地让婕斯泰（Geshtai）成了几乎和伊斯图丝一样重要的神祉。赞雅和汝欧肯仍然有其追随者。现代巴克鲁尼什王国的种子被播种到了土地上。（5110 SD / -406 CY）

2970 BH 一个来自奥瑞克遥远西方的部族穿过古老的巴克鲁尼什领土的南部，他们自称佩尼姆（Paynim）。其中一个部落，乌利（Uli）来到乌尔斯普鲁山脉（Ulsprue）与屏障峰之间的地区，建立起了乌尔（Ull）。（310 CY）

文化特性：最初的巴克鲁尼什是标准的游牧民族。虽然之后移居到了城市，但他们身上依然保留着一些野性。与苏人相比，他们的宗教信仰要简单得多。他们崇拜命运，崇拜广阔平原上的曙光，崇拜在身体和精神两方面的强韧。
巴克鲁尼什人和地球上的阿拉伯人十分相像，包括他们的文化。他们文明的最盛时期类似地球的波斯文明，而新兴的西部巴克鲁尼什国家则在对抗这种文明。

（Steve Wilson的巴克鲁尼什历史参考了Lenard Lakofka的苏人历史，以及Gary Gygax的成就，World of Greyhawk(TSR)，Dungeon Master’s Guild,1st ed.(TSR)以及Greyhawk Adventure(TSR)。）
